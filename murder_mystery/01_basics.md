# 01. マーダーミステリーの基本

**調査日: 2026-06-15**

---

## 1. マーダーミステリーとは何か

### 1-1. 定義

マーダーミステリー（通称「マダミス」）は、参加者が物語の登場人物となり、殺人事件などのシナリオの中で犯人を探し出す、あるいは犯人役として逃げ切ることを目指す、**会話劇型の推理パーティーゲーム**。

- 各プレイヤーはあらかじめ制作者が設計したキャラクターを割り当てられ、そのキャラクターとして行動・発言する
- 個人の「勝利条件（個人ミッション）」と「共通の真相解明」の両立を目指す
- **「一度しか遊べない」**ことが最大の特徴：一度プレイするとネタが分かってしまうため、同一シナリオは原則1回限り

### 1-2. 他ジャンルとの違い

| 項目 | マーダーミステリー | TRPG | ボードゲーム | 脱出ゲーム |
|------|------------------|------|-------------|-----------|
| キャラクター | 制作者が完成済み・固定 | プレイヤーが自作 | なし/薄い | なし/薄い |
| 繰り返しプレイ | 不可（1回限り） | 何度でも可 | 何度でも可 | 難しい（ネタバレ） |
| 目的 | 犯人特定 + 個人ミッション | 協力してミッション達成 | ゲームごとに異なる | 脱出・パズル解き |
| 雰囲気 | 緊張・心理戦・情報秘匿 | 協力・和気あいあい | ゲーム依存 | パズル・協力 |
| 自由度 | 限定的（情報コントロール中心） | 非常に高い | ルール内 | ルール内 |
| 所要時間 | 2〜3時間（完結） | 数時間〜数百時間 | 30分〜数時間 | 1時間前後 |

- **vs TRPG**: TRPGは「物語や世界をプレイヤーとGMで一緒に作っていく」のに対し、マダミスは「用意された事件とドラマの中に飛び込んで体験する」。
- **vs 脱出ゲーム**: 脱出ゲームはパズルを解いて脱出するのが目的。マダミスはキャラクターを演じながら殺人事件の真相を推理する点が根本的に異なる。

### 1-3. 起源・歴史

- **19世紀**: 推理小説の人気を背景に、欧米でディナーパーティー型ゲームとして萌芽
- **1948年**: ボードゲーム『クルード（Cluedo）』登場（クラシックな推理ボードゲームの原型）
- **1980年代**: 北米でTRPGブームに乗じて流行。1986〜87年に「Mafia」開発（現代の人狼ゲームの祖）
- **20世紀後半**: ヨーロッパで「マーダーミステリーパーティー」として普及（コスチュームを着るディナーイベント形式）
- **中国**: 「劇本殺（じゅほんさつ）」として独自進化。4万店舗超が存在する国民的趣味に

### 1-4. 日本での普及経緯

- **2018年頃**: 中国で流行していた「劇本殺」のシナリオが輸入される形で上陸
- その後、日本オリジナルシナリオが次々と制作・販売されるように
- YouTuberヒカル氏の動画配信が一般認知の拡大に貢献
- **コロナ禍（2020〜）**: 対面が制限される中、オンライン形式（Discord・UZUアプリ等）が急速に普及
- 現在は対面専門店が全国に約50店舗以上、スマホアプリ（UZU等）も展開

## 2. 基本的なゲーム進行の流れ

典型的なフロー（シナリオによって細部は異なる）。

| Phase | 名称 | 内容 |
|-------|------|------|
| 0 | 導入・セッティング | GMがルール説明、舞台・世界観・登場人物の関係を紹介 |
| 1 | 配役・HO読み込み | 担当キャラ（HO）が配られ、各自が背景・目的・秘密を把握。**他人のHOは読まない** |
| 2 | オープニング | GMが事件発生を読み上げ、全員が共通情報を受け取る |
| 3 | 第1議論（全体会議） | 全員での自己紹介・アリバイ確認・初期情報共有。全員に聞こえる発言のみ |
| 4 | 調査・情報公開 | カード／追加資料（証拠品・地図・証言文書）で新情報が公開され、交換・提示する |
| 5 | 密談 | 2〜3人単位の秘密会話。情報の選択的開示・取引・協力関係の構築 |
| 6 | 第2議論（全体会議） | 集めた情報・推理を持ち寄って議論。アクションフェーズを設ける場合も |
| 7 | 投票 | 「犯人だと思う人物」に記名投票。最多票が犯人候補として名指しされる |
| 8 | エンディング | GMが正解エンディングを読み上げ、ミッション達成度で点数集計。分岐することも |
| 9 | 感想戦 | 全員で振り返り。ここで初めて全HO・真相が共有され、ネタバレ解禁 |

## 3. 専門用語集

| 用語 | 定義 |
|------|------|
| **HO（ハンドアウト）** | 各キャラに配られる台本冊子。記憶・背景・秘密・目標・事件当日の行動が書かれた配布資料。自分のだけ読む |
| **GM（ゲームマスター）** | 進行役。シナリオ読み上げ・ルール説明・フェーズ管理・エンディング読み上げを担当。PLとしては参加しないことが多い |
| **PL（プレイヤー）** | キャラを演じる参加者 |
| **PC（プレイヤーキャラクター）** | PLが担当・演じる役柄 |
| **NPC** | PLが演じないキャラ。GMが声をあてたり書面上の存在として登場 |
| **RP（ロールプレイ）** | 現実の自分でなくキャラとして発言・行動すること |
| **全体会議** | 全員が集まって会話する議論フォーマット。密談はできない |
| **密談** | 2〜3人単位の秘密の話し合い。他PLには聞こえない形で情報交換・取引する |
| **調査フェーズ** | 証拠品・カード・追加資料が公開・配布され情報を集める段階 |
| **投票** | 「犯人と思う人物」に票を投じる。終盤に行い最多票が犯人候補に |
| **エンディング分岐** | 投票結果や特定アクションの有無で結末が変わる仕組み |
| **感想戦** | 終了後の振り返り時間。ここで全HO・秘密・真相が共有され自由に話せる |
| **勝利条件（個人ミッション）** | 各キャラの個人目標。犯人以外も犯人当て以外の目標を持つことがある |
| **共通ミッション** | 全PL共有の目標（例：真犯人を見つける） |
| **確白（かくしろ）** | 犯人でないことが公開情報として確定している状態 |
| **黒・白** | 「黒」=犯人っぽい、「白」=犯人でなさそう、という推理上の判断スラング |
| **卓** | 1回のプレイセッションを指す単位 |
| **オフセ / オンセ** | オフライン（対面）/ オンラインのセッション |
| **ネタバレ厳禁** | 結末・犯人・HO内容を未プレイ者に伝えないこと。界隈の最重要マナー |
| **メタ読み・メタプレイ** | 作中情報でなくゲーム外部視点（HO枚数等）で推理・発言すること。禁止行為 |
| **アリバイ** | 事件発生時刻に現場にいなかったことの証明 |

## 4. プレイ形態

### 4-1. プレイ場所

| 形態 | 特徴 |
|------|------|
| 対面・店舗型（オフセ） | 専門店でGM付き。臨場感・没入感が最高。全国に約50店舗以上 |
| 対面・パッケージ型 | 市販シナリオを購入し自宅等でプレイ。GMをPLが兼任することも |
| オンライン型（オンセ） | Discord等の通話＋ブラウザツール（ユドナリウム等）やアプリ（UZU）。コロナ禍以降に急成長 |
| アプリ型 | UZU等のスマホアプリ。入門として人気 |

### 4-2. 人数・時間

- **人数**: 3〜6人が最も多い。小規模（3人〜）から大規模（10人以上）まで存在。**1人欠けると成立しない**ことが多く、メンバー集めとドタキャン防止が重要
- **時間**: 短編30分〜1時間、標準2〜3時間、長編4時間以上。店舗開催は概ね3時間前後

### 4-3. 必要なもの

- **対面**: シナリオパッケージ（HO・証拠カード・GM資料）、テーブル、GM役（またはGM内包の店舗）
- **オンライン**: PC/スマホ、Discord等の通話ツール、ユドナリウム/ココフォリア等のブラウザツール（またはUZU）、シナリオ購入

## 5. プレイヤー体験の面白さ

| 観点 | 内容 |
|------|------|
| **没入感** | 第三者視点でなく登場人物の視点で物語に入り込む。当事者として物語を動かす能動性がある |
| **ロールプレイ** | 普段と違うキャラを演じる楽しさ。嘘をつく・感情表現する演技的快楽。初対面でも自然に交流できる |
| **推理** | 断片情報を組み合わせ全体像を浮かび上がらせる。トリックやアリバイの矛盾を見抜く論理の楽しさ |
| **駆け引き** | 情報を「いつ・誰に・どこまで」開示するかの戦略判断。犯人役のスリル、密談での取引・同盟 |
| **一回性** | 一度しか体験できないことが真剣さと緊張感を高める。「この体験は今だけ」の希少性 |
