## 3行サマリ

1. マダミスは「制作者が用意したキャラを演じ、情報の非対称性の中で犯人を推理する**一度しか遊べない会話劇**」。面白さの核は **没入・推理・駆け引き・一回性**。
2. シナリオは **HO（各自の秘密台本）＋共通情報＋フェーズ進行＋情報カード＋勝利条件＋GM台本** という部品でできており、設計の肝は「**誰が何を知っているか**（情報表）」と「**全員のタイムライン矛盾の排除**」。
3. 作り方の鉄則は **「結末から逆算」「凶器・タイムテーブル・動機の三軸を矛盾なく」「全員に物語と目標を持たせる」「最低3回テストプレイ」**。最大の難所はテストプレイと難易度調整。

## 自作に向けた次の一歩（メモ）

- まずは **1時間程度・4人前後の小規模シナリオ** から始めるのが現実的（テストプレイヤーを集めやすい）。
- 設計はテキストより **マトリクス（タイムテーブル表・情報表）** で管理すると矛盾が激減する（`03` §1-4, `02` §6 参照）。
- 配布形態（PDF / ウズスタジオ / ユドナリウム / ココフォリア / BOOTH）は作り始める前に決めておくと手戻りが少ない（`03` §7 参照）。
