# ココフォリア移行書 — 『覗きの代』ユドナリウム→CCFOLIA移植

作成: 2026-07-13（Web調査3系統＋公式ドキュメント精読の統合。仕様は2026-07時点）
対象読者: 別環境で移植作業を行う自分自身／Claudeセッション。**このファイル単体で作業に入れる**ことを目的とする。

関連: 図解版ガイド（Artifact） https://claude.ai/code/artifact/189896fb-01f3-44c2-8480-2a0c56879793

## 0. 要旨

- **ユドナリウム部屋データ（zip/XML）→ココフォリアの一括変換ツールは存在しない**（2026-07調査時点）。ココフォリアの部屋データ（zip内JSON）はスキーマ非公開のクローズド仕様で、`build_udon.py` のようなスクリプト直接生成は前提にできない
- したがって移植は「**PNG素材を流用し、ココフォリア上で部屋を手で組み直す**」方式。機能対応はすべて揃っており詰む要素はない
- 重要な制約は3つ:
  1. **部屋データのエクスポート（配布・バックアップ・部屋複製）は有料の CCFOLIA PRO 限定**（月500円=2023年時点価格、現行要確認）。インポートは無料ユーザーでも可
  2. **1MB超の画像はアップロード時に自動で1MB以下へ強制圧縮**（2025-08仕様変更）。手元素材15枚が該当 → §4-A で事前圧縮
  3. **秘話・プライベートチャットはルームマスター（GM）がログを閲覧できる**仕様。PL間の完全秘匿は不可 → 密談はDiscord VC分割で運用

## 1. 前提知識（ココフォリアの基本）

### 1.1 サービス形態・料金

- ブラウザ完結のオンセツール。公式サーバーのみ（自前ホスティング不可）。https://ccfolia.com
- 無料（BASIC）: 部屋作成・プレイ・部屋データの**インポート**まで可能。部屋数上限は現在なし
- CCFOLIA PRO（月額500円・2023年時点。現行価格は要確認）:
  - **ルームデータのZIPエクスポート**（←配布とバックアップに必須）
  - 公開リンク発行／ルーム複製／タイマー／Audiostock音源／広告非表示
- 制作者側だけPROに入ればよい。PLは無料アカウントで部屋URLから入室

### 1.2 部屋データと配布規約

- 配布の標準形: BOOTH等で「部屋データZIP同梱→購入者がドラッグ＆ドロップでインポート」
- **ZIPは展開せずそのまま**インポートに使う（展開するとエラー）
- **音源（BGM/SE）はエクスポートZIPに含まれない**。配布時は別導線が必要
- 規約（ccfolia.com/termsOfService.html）: 部屋データの再配布・転載は営利/非営利問わず禁止。他人の部屋・公開リンク経由の部屋は複製・エクスポート不可

### 1.3 画像の制約

- 対応形式: **JPG / PNG / GIF / APNG**（WebP非対応との情報あり）
- **1MB超は自動で1MB以下に強制圧縮**（2025-08〜）。文字の細かい論文カードが劣化するため**事前に自前で1MB未満に最適化するのが必須**
- 推奨サイズの公式明記はなし。コミュニティ慣習: 背景1200×900px前後、パネルは幅1マス=24px換算
- 画像リスト表示上限3,000点

## 2. UI・操作リファレンス

### 2.1 画面構成

```
┌────────────────────────────────────────────────┐
│ [ルーム名] [BGM01][BGM02]   （右上）シナリオテキスト/シーン一覧/    │ ←上部バー
│                          カットイン一覧/ｽｸﾘｰﾝﾊﾟﾈﾙ一覧/           │
│                          ﾏｰｶｰﾊﾟﾈﾙ一覧/ﾏｲｷｬﾗｸﾀｰ一覧              │
├───────────────────────────────┬────────────────┤
│                               │ 盤面キャラクター一覧 │
│    盤面（背景・前景＋パネル＋コマ）      │ （ｽﾃｰﾀｽ/行動順）    │
│    ドラッグ移動・ホイール拡縮            │                │
├───────────────────────────────┼────────────────┤
│ チャット欄（タブ: メイン/情報/雑談…）      │ 右下: 前景背景変更/  │
│                               │ パネル追加/キャラ選択 │
└───────────────────────────────┴────────────────┘
```

- **ルーム名クリック → ルーム設定**（メンバーリスト=権限変更、セーブ/ロード、ルームデータ出力）
- 表示範囲は各ユーザー独立（自分がスクロールしても他人の画面は動かない）

### 2.2 基本操作の原則

- **右クリック＝操作メニュー、ダブルクリック＝編集ウィンドウ**でほぼ一貫（スマホは長押し）
- 画像登録はどの機能でも共通の「画像選択ウィンドウ」に**ドラッグ＆ドロップ**。複数一括可・タブで用途別整理可・「NOIMAGE」で画像なし化
- 部屋作成: マイルーム一覧右下「＋」→ ルーム名とダイスボットのみ設定（マダミスは判定不要なので「DiceBot」のままでよい）

### 2.3 盤面に置ける3種のオブジェクト

| | スクリーンパネル | マーカーパネル | キャラクターコマ |
|---|---|---|---|
| シーン切替時 | **残り続ける** | **そのシーン専用（切替で消える）** | 残る（「しまう」で退場可） |
| 裏面画像 | ○（カード化できる） | × | — |
| 公開制御 | ○ 4モード（§2.4） | ×（常に全員に見える） | ステータス非公開等のトグルあり |
| 主用途 | **証拠カード・HO・常設掲示** | **シーン連動の演出画像・段階公開** | PC/NPC/探索マーカー |
| 作成 | 上部「スクリーンパネル一覧」＋ / 盤面右クリック | 上部「マーカーパネル一覧」＋ / 盤面右クリック | 右上「マイキャラクター一覧」＋→「盤面に追加」 |

### 2.4 スクリーンパネル詳解（マダミスの主戦場）

編集ウィンドウの設定項目: 横幅・縦幅（1マス=24px）／重なり優先度／メモ（**ホバーで表示**→カード要旨を仕込むと便利）／**配置固定**／**サイズ固定**／地形化／画像（表面）／**背面画像（裏面）**／高度な設定（クリック時アクション: チャット送信・ダイス・シーン遷移・PDF表示）

**公開4モード（右クリックで切替）**:

| モード | 見え方 | 用途 |
|---|---|---|
| 非公開にする | 全員に裏面 | 未発見の証拠カードの初期状態 |
| 全体に公開する | 全員に表面 | 手がかり公開の瞬間 |
| 自分だけ見る | 本人のみ表面。他PLには「○○が見ています」表示 | **個別HO・キャラシの定石** |
| 自分以外に公開する | 本人以外に表面（v1.31追加。パネル選択→Wキー） | 出題側だけ見えない演出 |

- **配置固定・サイズ固定は必ずON**（PLの誤ドラッグでカードが動く/伸びる事故が定番）
- 山札: 同サイズパネルを重ねて右クリック→「山札を作る」。ただし**引く順序は常にランダム**（積み込み不可）。本作の証拠カードはGMが段階配置する方式なので**山札化せず個別パネル**で組む

### 2.5 シーン（GM専用）— フェーズ進行の骨格

- 「前景＋背景＋BGM＋マーカーパネル」のセットを丸ごと保存し、サムネイルクリックで一括転換（BGM無し/切替テキスト無し反映も選択可）
- 作成: 盤面を望む状態に整える → 右上「シーン一覧」→「＋」→ **いまの盤面がそのまま保存される**
- 修正は「盤面を保存」で**丸ごと上書きのみ**（シーン内要素の個別編集は不可）
- **墨消し→完全版の段階公開はこれで自動化**: 前半フェーズのシーンに墨消し版マーカーパネル、後半シーンに完全版を貼る

### 2.6 チャット・ダイス

- デフォルトタブ「メイン/情報/雑談」（削除不可）。タブ追加はGMのみ。追加時「プライベート」を選ぶと指定メンバー限定タブ（南京錠マーク）
- **秘話**: 相手の発言ホバー→ボタン→2人専用タブ自動生成（PLも可）
- **秘話もプライベートタブもGMはログ閲覧可**。GM進行の本作では監視できて好都合だが、PL同士の完全秘匿は不可
- ダイス: `1d100`、判定 `1d100<=50`（青=成功/赤=失敗）、**シークレット `s1d100`**（本人のみ表示→後から公開ボタンで開示）
- ログはHTML出力可（全タブ一括はGMのみ）

### 2.7 BGM・カットイン

- BGM: 上部バー「BGM01」「BGM02」の2トラック同時再生（環境音＋曲）。音量は音源側設定（全員反映）と個人設定の2層
- カットイン（GM専用）: 上部「カットイン一覧」＋で画像＋音声の割り込み演出を登録。**チャット末尾にカットインのタイトルを書いて送信すると発動**。NOIMAGE登録で実質SE再生ボタン
- ルーム変数＋カットインの組み合わせでED分岐演出の自動化事例あり（参考: 「純血種は語れない」部屋解説）

### 2.8 権限

ルーム設定→メンバーリストで5段階を割当: **ルームマスター／サブマスター／プレイヤー／見学／アクセス不可**

- プレイヤー権限でできる: キャラ作成・自コマ操作・パネルの公開切替（めくる/自分だけ見る）・山札から引く・秘話・シークレットダイス
- できない（GM/サブマスターのみ）: パネル新規作成・シーン切替・カットイン登録・チャットタブ編集・ルーム設定

## 4. 移植作業手順

### Phase A: 素材準備（他環境でも最初にやる）

1. **1MB超PNGの圧縮**（強制圧縮による文字潰れ回避）。2026-07-13計測で対象15枚:

   | ファイル（udonarium_test/outputs/） | MB |
   |---|---|
   | design/final_backdrop.png | 1.77 |
   | design/final_board_zones.png | 1.70 |
   | design/final_board.png | 1.36 |
   | p1_all.png | 1.35 |
   | test_paper2_full.png | 1.31 |
   | p1_final.png | 1.23 |
   | test_paper2_redacted.png | 1.20 |
   | design/final_prologue.png | 1.19 |
   | design/final_paper_2_full.png | 1.16 |
   | design/final_paper_3_full.png | 1.12 |
   | design/final_paper_4_full.png | 1.11 |
   | design/final_paper_1_full.png | 1.10 |
   | design/final_rules.png | 1.08 |
   | design/final_paper_2_redacted.png | 1.06 |
   | design/final_paper_1_redacted.png | 1.01 |

   ※ `test_*` / `p1_all` / `p1_final` は中間生成物の可能性が高いので、配布対象は `design/final_*` を優先確認
   方法: 高品質JPEG化（品質85前後）または減色PNG。文字の可読性を目視確認しながら `ccfolia/assets/` に出力する（原本は残す）
2. 出力先フォルダ案: `ccfolia/assets/`（圧縮済み・アップロード用一式）。ファイル名は原本と同名を維持

### Phase B: 部屋構築（ココフォリア上の手作業）

1. アカウント作成（無料）→部屋作成（ルーム名=シナリオ名、ダイスボット=DiceBotのまま）
2. 背景・パネル用画像を画像選択ウィンドウへD&D（用途別タブで整理: 背景/カード/キャラシ/演出）
3. 常設スクリーンパネル配置: ルール・フェーズバー・行動マトリクス（全体公開・配置固定）
4. 証拠カード20枚＋扉カードをスクリーンパネルで配置（背面画像設定→「非公開」→配置固定・サイズ固定→メモ欄に要旨）
5. キャラシ初期版4枚を配置（非公開。セッション開始時に各PLが「自分だけ見る」）
6. キャラクターコマ作成: PC4人＋探索マーカー4種＋NPC（NPCは「しまう」で待機）
7. シーン構築: Phase0から順に「背景＋BGM＋そのフェーズのマーカーパネル（論文の墨消し/完全版）」を整えて「＋」で保存 ×Phase0〜9
8. チャットタブ整備: 必要なら個別連絡用プライベートタブ×4（GM⇔各PL）
9. BGM/SE・カットイン登録（使う場合。音源は配布ZIPに入らない点に注意）

### Phase C: 検証（GMリハーサル）

- [ ] 別アカウント（or 未ログインブラウザ）でPL視点入室し、非公開カードの見え方・「自分だけ見る」の表示を確認
- [ ] シーンをPhase0→9まで順に切替え、論文の墨消し→完全版が正しく入れ替わるか確認
- [ ] パネルの配置固定漏れがないか一通りドラッグして確認
- [ ] GM進行台本（`dist/GM専用/01_GM進行台本.md`）と突き合わせて公開タイミングのリハーサル

### Phase D: 配布・バックアップ（必要になったら）

- CCFOLIA PROに加入 → ルーム設定からZIPエクスポート
- 検証: エクスポートZIPを別部屋（無料アカウント側）にインポートして完全再現されるか確認
- 頒布時はBOOTH標準形式（ZIP同梱）。規約の再配布禁止条項に留意

## 5. 未確認事項（作業時に現地確認）
