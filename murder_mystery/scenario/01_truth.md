## 1. 真相の全体像

この施設（○○県御影村██████研究所）には、**人ならざる上位存在（同種）が2体**関わっている。

- **善玉A（民俗学者）**: 成熟を全うした個体。自前の身体を持ち、人間を壊さない。ルーツを知るため再訪した
- **悪玉B（ジャーナリスト ← 先遣隊の器 ← 封印容器）**: 成熟を飛ばした個体。自前の身体を持てず、覗くと相手を壊す。知的飢餓に駆られて外界を目指す

両者は「人を分かりたい／自分を知りたい」という同じ渇きを持つが、**成熟度の違いが善悪を分ける**。

| | 善玉A（民俗学者） | 悪玉B（ジャーナリスト） |
|---|---|---|
| 成熟 | 全うした（自前の肉体と自己意識を獲得） | 飛ばした（探求力だけが肥大） |
| 覗き | 共鳴経路（人格を保つ・共存可能） | 破壊経路（求心力を奪い崩壊させる） |
| 器 | 自前の身体（誰も殺していない） | 他人の身体を奪う（ジャーナリストの魂を抜いた） |
| 動機 | ルーツ探求（起源を知りたい） | 知的飢餓（もっと多くの人間に触れたい） |
| 人間への害 | なし | あり（覗く＝崩壊＝殺害に等しい） |

## 2. 完全時系列

### 【過去】施設運用期

1. ○○県御影村に、上位存在を研究する施設が建設される
2. **個体α（＝A）が人間に擬態して施設に侵入**。研究者と協力的に接触。共存可能と判断される
3. **個体β（＝B）が施設に侵入**。激しい抵抗。制圧のうえ**封印容器に収容**される
4. 施設は2体を研究。覗き込み現象に「2系統の反応パターン」（共鳴／破壊）があることを観察
5. **個体αが脱走**。追跡チームが展開するも確保に至らず。捜索打ち切り
6. 施設は最終的に放棄される（経緯の詳細は未確定）。個体βは容器に封印されたまま残される

### 【先遣隊】事件

7. 何年か後、先遣隊（調査チーム3名）が施設に派遣される。調査隊員の同僚が含まれる
8. 先遣隊が施設内を調査。**事故的にBの封印が緩む**
9. Bが容器から脱出。先遣隊と接触し、**全員を覗く → 求心力を吸収し、奪った力で隊員の姿を模倣**。しかし奪った器は長く持たず灰化するため次々と乗り換える → **先遣隊全員が崩壊**（覗かれただけの隊員＝かろうじて人の形だが身元特定不可 / 器として使い切られた隊員＝灰のみ）
10. 覗きを通じてBは**先遣隊の高度な知性・外部知識を獲得**。先遣隊の外部への報告は**完全に途絶**

### 【封鎖】空白期間

11. 先遣隊の通信が途絶。**施設の外周セキュリティが自動発動し、施設が封鎖される**
12. Bは施設内に閉じ込められる。封鎖中に施設内を探索
13. Bは**封印装置の存在**と**「封印完了＝封鎖解除」の条件**を把握する
14. しかし一人では脱出できない（封印対象が必要＋外周封鎖は内側からは解除不可）
15. **Bは自発的に封印容器に戻る**。いつでも出られる状態で、次のチームの到着を待つ

### 【現在】ゲーム開始

16. 正体不明の施設に通常の救助班では対処困難と判断、各分野の専門家を含む第二次調査チーム（PL4名）が編成される。メンバー: 調査隊員、大学教授、科学ジャーナリストの正規3名 ＋ **民俗学者（＝A）がデータを改ざんして4人目として紛れ込む**
17. チームが施設に降下。**出口が封鎖される**（施設は先遣隊事件以降ずっと封鎖状態。チーム降下で再封鎖）
18. **Bが容器内でチームの到着を検知し、容器から出る**

### 【ゲーム中】Phase別

| Phase | 裏で起きていること |
|-------|-----------------|
| **0. 導入** | 全員が降下。先遣隊の崩壊を目撃。Aは「引き寄せられて来た」ことに違和感を覚えつつ、民俗学者として振る舞う。Bはまだ器に入る前（先遣隊の残った器 or 独立状態で潜伏中） |
| **1. 第1調査** | 施設の表層を探索。「化け物がいた施設」が分かる。Bはこの時点ではまだ行動していない |

| **4. 第2調査** | 隔離区画へ。灰の残骸の発見＝なりかわりの物証。自動認証扉の反応テスト（上位存在がいれば開く）。金庫（論文①）。封鎖プロトコル通知 |

| **7. 最終会議＋投票** | 脱出手段の提示→推理発表→自由議論→投票→封印 |
| **8. エンディング** | 結果で分岐（下記） |

## 3. 善玉A（民俗学者）の真相

### 正体

成熟を全うした上位存在。自前の肉体と自己意識を持つ。覗きは共鳴経路を取り、人間を壊さない。

### 来歴

- 過去に施設に侵入し、研究者と協力的に接触した（個体α）
- 施設から脱走。以後は人間社会に溶け込んで生活
- 自分の起源（なぜ存在するのか、何者なのか）を知りたいという渇きに駆られ、施設に再訪
- 調査チームのデータを改ざんし、4人目の民俗学者として紛れ込んだ

### ゲーム開始時の状態

- **自分が上位存在であることを知らない**。ただし「なぜ自分はここに引き寄せられたのか」「なぜデータを改ざんしてまで来たのか」という違和感はある
- Phase 3の固有ターン（想起）で正体を自覚する

### 想起後に得る知識

| 知っていること | 知らないこと |
|-------------|------------|
| 自分は上位存在である | Bが誰に擬態しているか |
| 過去にこの施設に来たことがある | Bの具体的な能力（覗き＝破壊） |
| 封印装置が最奥区画にある | 自動認証扉の開錠条件の詳細 |
| 封印の手順（断片的）| |
| もう1体の同種がこの場にいる（求心力の知覚） | |
| データ改ざんの事実 | |

### 勝利条件

| 段階 | 条件 |
|------|------|
| 初期（Phase 0〜3前半） | ①施設の脱出手段を調査する ②記憶の手がかりを探す ③データ改ざんの事実を秘匿する |
| 想起後（Phase 3後半〜） | ①同種を特定し、投票で封印に票を集める ②自身の正体を秘匿する ③全員で生きて脱出する |

### PLから見た疑惑ポイント（罠①＝最濃の赤いニシン）

- 「本来3名のところ4人目」＝紛れ込んだ者として最も疑われやすい
- 施設に詳しすぎる（過去の訪問歴）
- 逃走記録が発見される（P7）
- 自動認証扉が反応する（上位存在だから）
- **しかし実際は善＝民俗学者を封印してもBは外に出る**

## 4. 悪玉B（科学ジャーナリスト）の真相

### 正体

成熟を飛ばした上位存在。自前の肉体を持てず、他者の器を奪って擬態する。覗きは破壊経路を取り、相手の求心力を奪い崩壊させる。

### 来歴

1. 過去に施設へ擬態侵入 → 特性ゆえ捕獲・封印（容器に収容）
2. 先遣隊の侵入で封印が緩む → 容器から脱出 → 先遣隊全員を覗く
3. 覗きを通じて先遣隊の高度な知性・外部知識を獲得。**奪った求心力で隊員の姿を模倣**するが、器は長く持たず灰化 → 乗り換えを繰り返す → 先遣隊は全員崩壊（覗かれただけ＝かろうじて人の形 / 器として使用＝灰のみ）
4. 先遣隊の通信途絶（報告は完全に途絶）→ 施設が自動封鎖 → Bは施設内に閉じ込められる
5. 封鎖中に施設内を探索 → 封印装置の存在・「封印完了＝封鎖解除」の条件を把握
6. 自発的に封印容器に戻る（いつでも出られる状態で待機）
7. 第二次チーム到着を検知 → 容器から出る
8. Phase 3でジャーナリストにオンタイム乗り換え

### Bの模倣能力

- 覗きで求心力を吸収した後、**奪った力を元に対象の姿を模倣できる**
- ただし奪った器は長く持たず灰化するため、定期的に乗り換えが必要
- 先遣隊で複数の灰が残っているのはこのサイクルの結果
- ジャーナリストには「器ごと入れ替わった」状態（模倣ではなく身体そのものを乗っ取り）

### ゲーム開始時の状態

- Phase 0〜2では**ジャーナリストは純粋な人間**。PLは何も知らずに「科学ジャーナリスト」として行動する
- Phase 3の固有ターンで**Bがオンタイムで乗り換え**。PLは追加HOを受け取り、目標が全面的に切り替わる

### 乗り換え後に得る知識

| 知っていること | 知らないこと |
|-------------|------------|
| 自分は上位存在である | Aが誰に擬態しているか |
| 過去にこの施設に囚われ封印されていた | 自動認証扉の開錠条件 |
| 先遣隊を覗いて崩壊させた＋その知識を吸収した | 封印の正確な手順（理論面は教授の研究知識が必要） |
| もう1体の同種がこの場にいる（求心力の知覚） | |
| 封印装置が最奥区画にある（封鎖中の施設探索で発見） | |
| 封印完了＝施設封鎖解除（施設の文書・設備から把握） | |
| **認証扉の仕組み**（空洞検知＝求心力のない存在を検知する機構。施設探索で把握） | |
| **過去に1体が脱走した記録がある**（施設の保安記録から把握） | |

### 勝利条件

| 段階 | 条件 |
|------|------|
| 初期（Phase 0〜3前半） | ①教授の理論を自分なりに検証する ②施設の現象を自分の目で確かめる ③脱出手段を見つける |
| 乗り換え後（Phase 3後半〜） | ①正体を秘匿する ②施設から脱出する（＝自分以外の誰かを封印させる） |

### Bの最適戦略

- **「民俗学者＝紛れ込んだ奴＝危険」と全力で誘導する**（罠①を利用）
- Aを封印させれば脱出成立（Aも上位存在なので封印→ロック解除）
- 人間を封印させても脱出成立（ただし最悪の結末＝Bが外に出て人間社会が危険に）
- **弁明の武器**: 取材者としての中立性＋Phase 0〜2で「人間だった」実績

## 5. 人間2人の立場

### 調査隊員

- 正規メンバー3名の1人。先遣隊に同僚がいた
- 開錠スキルで金庫を開けられる＝証拠に最も多く触れる
- **濡れ衣リスク（中）**: 出歩いていた＋証拠に詳しすぎる＝「なぜそんなに知っている？」

### 大学教授

- 正規メンバー3名の1人
- 査読スキルで研究資料の墨消しを読める＝理論面の鍵を握る
- **濡れ衣リスク（中）**: 研究資料に自分の名前が引用元として載っている＝「施設と関係者？」
- 勝利条件: 研究資料の入手＋元凶排除＋生きて脱出

## 6. 証拠の対応表（消去法格子）

### 特定の本筋＝3つの物証の交点

```
物証1: 認証扉テスト → 上位存在 = {民俗学者, ジャーナリスト}
物証2: 崩壊痕跡＋研究 → 「入れ替わりには単独の隙が要る」
物証3: 出歩き → 隙を持っていた = {ジャーナリスト, 調査隊員}

交点: 上位存在 ∩ 出歩き = ジャーナリスト = B
```

| 物証 | 割る集合 | 供給元 | Phase |
|------|---------|--------|-------|
| **認証扉テスト** | 上位存在2人（民俗学者・ジャーナリスト）vs 人間2人（調査隊員・教授） | 自動認証扉（固有マス札）×2回 | P4・P7 |
| **崩壊痕跡＋研究** | 「入れ替わった者がいる」（隙が必要） | 灰の残骸（P4）＋論文①②③ | P4・P7 |
| **出歩き** | 隙を持っていた2人（ジャーナリスト・調査隊員） | 固有ターン1で目撃情報として開示 | P3 |

### 赤いニシン（誤誘導）3枚

| 罠 | 対象 | 仕掛け | 実態 |
|----|------|--------|------|
| **罠①（最濃）** | 民俗学者(A) | 紛れ込み＋施設に詳しい＋逃走記録＋扉が反応 | 善玉。封印しても脱出は成立するがBが外へ |
| **罠②（中）** | 調査隊員 | 出歩き＋証拠に最も触れる＝「なぜ知っている？」 | 人間。無実 |
| **罠③（中）** | 教授 | 研究に自分の名前が引用元＝「施設の関係者？」 | 人間。無実。査読するほど疑惑が増す |

### 罠の反転

論文③（P7・金庫）が**成熟モデル**を開示:
- 「脱走した個体＝成熟済＝共鳴＝**保つ側（善）**」
- 「収容されていた個体＝未成熟＝破壊＝**壊す側（悪）**」

→ 「逃げた奴が危険」という直感が覆る。**逃げた方が善、残された方が悪**。

## 8. エンディング分岐

### 封印プロセス（共通）

投票で最多票の人物が決定した後、**封印装置を装着するまでの過程をPLがロールプレイする**。全員で対象を最奥の封印装置まで連れて行き、装置を起動する。

### 封印結果の分岐（対象が誰かで3分岐）

| 封印対象 | 結果 | 封印時の描写 | 脱出後の描写 |
|---------|------|------------|------------|
| **悪玉B（ジャーナリスト）** | **正解エンド** | 装置が起動。ジャーナリストの身体が**灰になって崩れ落ちる**——上位存在の器が求心力を失い崩壊する光景。Bの知的飢餓は永遠に封じられる | 施設封鎖が解除。残り3人で脱出 |
| **善玉A（民俗学者）** | **B勝利エンド** | 装置が起動。民俗学者の身体が**灰になって崩れ落ちる**。善玉の器もまた、上位存在ゆえに灰化する | 施設封鎖が解除。3人＋B（ジャーナリストのふり）で脱出。**Bが人間のふりをして外界へ**——知的飢餓を抱えた破壊的存在が社会に放たれる（後日談は下記） |

### Aの処遇（サブエンディング・2026-06-30 追加）

メイン封印の**後**、**調査隊員が生存している場合**に発生するサブエンディング。調査隊員の初期任務「施設関係者の特定・確保」の回収。調査隊員が封印された場合はスキップ。

**処理**: GMが調査隊員に個別質問（「この中に施設関係者だと思われる人物はいますか？」→ 指名した場合「確保しますか？」）。判定は**調査隊員の単独判断**。

| パターン | 調査隊員の判断 | 結果 |
|---------|--------------|------|
| **A) 正しく特定＋確保** | 民俗学者を施設関係者と特定し、確保する | Aは調査本部の隊員により連行される。任務完遂 |
| **B) 正しく特定＋解放** | 民俗学者を特定するが、無害と判断して報告しない | Aは自由の身。調査隊員は任務を果たさない選択をした |
| **C) 誤認特定＋確保** | 大学教授を施設関係者と誤認し確保 | 無実の教授が連行される。Aは見逃される（※正解エンドでのみ発生） |
| **D) 誰も特定しない** | 「分からない」「いない」と判断 | Aは判明しないまま全員で脱出 |

> **設計意図**: テストプレイで「Aが安易に正体を明かして味方に付く」が発生したため、**正体開示＝確保リスク**のジレンマを追加。Aにとって「正体を明かしてBに投票を集める」戦略が**確保されるリスクと天秤**になる。

**処理手順**: エンディング描写の**前**に、調査隊員PLに**非公開で**確認（「施設関係者だと思われる人物はいますか？」→ 指名した場合「確保しますか？」）。パターン確定後、以下の共通冒頭を読み、調査隊員PLに本部への報告をRPしてもらう。RP後、対応するパターンの続きを読む。
