# 大学院生（悪玉B）── ②人ごとビートシート

> **位置づけ**: `02_beatsheet.md`（①全体フロー）に対する大学院生固有の差分。
> HOの素材として使う。

## 前提

教授のゼミ生。認知科学専攻。フィールドワーク経験あり、教授に誘われて参加。**Phase 0〜2 では純粋な人間**。固有ターン1（Phase 3）でBがオンタイムで乗り換え、以降は悪玉Bとなる。PLは乗り換え時に追加HOを受け取り、目標が切り替わる。

**キャスティング**: 役職名のみで選ぶ（正体・変化は一切告知しない。完全サプライズ）。

### Bの来歴

1. 過去に施設へ擬態侵入 → 特性ゆえ捕獲・封印（容器に収容）
2. 先遣隊の侵入で封印が緩む → 容器から脱出 → 先遣隊全員を覗く
3. **覗きを通じて先遣隊の高度な知性・外部知識を獲得** → 先遣隊は全員崩壊
4. 先遣隊の通信途絶 → **施設が自動封鎖（外周封鎖発動）** → Bは施設内に閉じ込められる
5. 封鎖中に施設内を探索 → **封印装置の存在・「封印完了＝封鎖解除」の条件を把握**
6. 外界に出ればより多くの人間に触れられる＝**知的飢餓が動機**。だが封鎖中は脱出不可能
7. **自発的に封印容器に戻る**（いつでも出られる状態で待機）── 次のチームの到着を待つ
8. 第二次チーム到着を検知 → 容器から出る → Phase 3で院生に**オンタイム乗り換え**

### Bが乗り換え後に得る記憶・知識

| 知っていること | 知らないこと |
|-------------|------------|
| 自分は上位存在である | **Aが誰に擬態しているか** |
| 過去にこの施設に囚われ封印されていた | 自動認証扉の開錠条件 |
| 先遣隊を覗いて崩壊させた＋その知識を吸収した | 封印の正確な手順（理論面は教授の研究知識が必要） |
| **もう1体の同種がこの場にいる**（求心力の知覚） | |
| **封印装置が最奥区画にある**（封鎖中の施設探索で発見） | |
| **封印完了＝施設封鎖解除**（施設の文書・設備から把握） | |

> ※旧設計の「前の器の場所を知っている」は削除。崩壊度の差＋装備差で誰でも気づける設計のため、Bの特権情報にする必要なし。

### 求心力の知覚（民俗学ベース）

Bが覚醒した瞬間、器を通じた「見え方」が変わる。人間には求心力（魂の重み）があるが、**1人だけ自分と同じように空洞の存在がいる**。ただし視界が不安定なため**誰かまでは特定できない**。

- 追加HOに記載:「この中にもう1体、自分と同じ存在がいると感じる。誰かは分からない」
- **Aにも同じ知覚がある**（対称設計）
- 「異形は同類を感じ取る」民俗学モチーフに準拠

## 勝利条件（2段階）

| 段階 | 条件 |
|------|------|
| **初期**（Phase 0〜3前半） | ①教授の理論を自分なりに検証する ②施設の現象を自分の目で確かめる ③脱出手段を見つける |
| **乗り換え後**（Phase 3後半〜） | ①**正体を秘匿する** ②**施設から脱出する**（＝自分以外の誰かを封印させる） |

### 初期目標の設計意図

人間フェーズでの個人的動機＝**教授への依存から自立したい知的好奇心**。認知科学の仮説を自分で検証したい野心。これがBの知的飢餓と二重写しになる。

### 脱出メカニクス（全員共通）

| 手段 | 条件 | Bにとっての意味 |
|------|------|---------------|
| **封印** | 誰かを最奥の封印装置で封印（意識抽出→廃人化）→ ロック解除 | **自分以外の誰かを封印させれば脱出できる**。Aでも人間でもよい |
| **救助待ち** | 封印しない／できない場合 → いずれ救助隊が来る（時間不明） | 脱出は可能だが、時間経過で器が劣化＋バレるリスク |

### 封印装置の仕様

- **依代ごと意識を抽出・格納する装置**（頭に嵌める電極デバイス）
- 上位存在専用ではない。**人間に使用しても同様に意識が抜かれ廃人化する**
- 民俗学的な「魂抜き」を科学で再現した装置
- 覗き（求心力喪失→身体崩壊）とは別の機序：封印装置は意識を抽出するため**身体は残るが廃人**

→ Bの最適戦略は**「自分以外の誰かに嫌疑を向けて封印させる」**。Aを売るのが最善だが、人間を売っても脱出は成立する（ただし本当の脅威＝Bが外に出る最悪の結末）。

## 固有スキル

### 表の能力（キャラシートに記載・他PLにも見える）
**「未閲覧の公開カードを確認できる」** — フィールドにあるまだ誰も読んでいないカードを見る。無害に見えるユーティリティスキル。固有ターン限定。

### 裏の能力（追加HOで開示・乗り換え後に使用）
**「対象の記憶を断片的に覗く」** — 人を指定。前フェーズのバディ調査でその人が得た証拠品1つを閲覧。各固有ターン1回＝最大2回。

### 処理手順（GM個別ヒアリング）

```
GM → 院生を個別に呼ぶ（固有ターンは全員順番に個別対応）
  → 「覗ける対象は〇〇と〇〇です」と案内
  → 院生が人を指定
  → GMがその人の前フェーズの証拠品1つを非公開で開示
  → 戻す
```

他PLには何をしたか見えない。聞かれたら「公開カードを確認した」で通る。

### テーマとの接続

「人の記憶を覗く」＝Bの能力（覗きで知識を吸収する）の弱い版。知的飢餓の動機とメカニクスが一貫する。

## Phase別

| Phase | 本人の主観・固有イベント | 推奨行動 | ジレンマ |
|-------|------------------------|---------|---------|
| **0. 導入** | 教授と共に施設に降りる。崩壊した先遣隊を目撃 | 状況把握。自分なりの観察を始める | なし（純粋な人間） |
| **1. 第1調査** | バディペアで施設探索。教授の知識を借りつつ自分の目で検証したい | 手がかり収集。教授とは独立した視点で情報を集めようとする | なし（まだ人間。知的好奇心は本物） |
| **2. 第1会議** | 「人知を超えた存在がいる」と全体が認識 | 情報共有に参加。認知科学の知識で解釈を試みる | なし（まだ人間） |
| **3. 固有ターン1** | **Bがオンタイムで乗り換え**。追加HOで目標全面切替＋求心力の知覚。**覗き見1回目** | 覗き見で前フェーズの証拠品を1つ確認（自分に不利な証拠 or Aの手がかり） | Phase 0〜2の「普通の自分」と矛盾する言動を取れない |
| **4. 第2調査** | Bとして行動開始。崩壊度の差（装備差）が発見される可能性。自動認証扉に触れると偶然開く可能性 | 脱出に繋がる情報を探す。Aの正体を推理。疑われないよう調査に協力する振り | 自動認証扉＝暴露リスク。崩壊度の差について詳しすぎる反応をすると怪しい |
| **5. 固有ターン2** | **覗き見2回目** | 封印手順の進捗を偵察 or Aの特定材料を探る | 何を見たか説明を求められる可能性 |
| **6. 最終調査** | 脱出とA特定を同時進行。Phase 3前後の言動変化が痕跡として浮上する可能性 | Aを特定し封印対象として根拠を集める。自分への疑惑を逸らす | 積極的に動く＝疑われる。消極的＝脱出が遠のく |
| **7. 最終会議＋投票** | **全員投票で封印対象を決定**。弁明の武器＝教授との師弟関係＋Phase 0〜2の「人間だった実績」 | 自分以外に票を集める。民俗学者＝「4人目、施設に詳しすぎる」。教授の信頼を利用 | 教授を利用＝師弟関係の裏切り。Aが反論してくる |
| **8. エンディング** | 投票結果で分岐。全員で対象を最奥の封印装置まで連れて行く | — | — |

### エンディング分岐（B視点）

| 結果 | Bの勝敗 | 描写 |
|------|--------|------|
| **B以外が封印される** | **勝利** | Bは人間のふりをして脱出。知的飢餓を抱えたまま外界へ |
| **Bが封印される** | **敗北** | 意識を装置に抽出される。院生の身体は廃人化 |
| **封印失敗（投票不成立等）** | **不確定** | 救助待ち。時間経過でバレるリスク＋器の劣化 |
| **バッドエンド（封印装置に言及ゼロ）** | **全員敗北** | 脱出方法を見つけられず施設に閉じ込められる |

## 比重チェック

- **Phase 0〜2**: 知的好奇心による自発的な調査。この間の行動が乗り換え後の偽装基盤
- **Phase 3**: 最重。オンタイム乗り換え＋目標切替＋求心力知覚＋覗き見1回目
- **Phase 4〜6**: 「嫌疑誘導＋脱出」の暗闘。崩壊度の差の発見が転機
- **Phase 7**: 投票＝最終勝負。師弟関係を武器にした弁明 vs Aからの告発
