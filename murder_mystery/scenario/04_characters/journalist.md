## 前提

大手科学雑誌の記者。先遣隊失踪事件のスクープを追い、同行取材枠として正規メンバーに加わった。取材現場でのフィールド経験が豊富で、科学記事の執筆を通じて研究資料を読み解く素地がある。**Phase 0〜2 では純粋な人間**。固有ターン1（Phase 3）でBがオンタイムで乗り換え、以降は悪玉Bとなる。PLは乗り換え時に追加HOを受け取り、目標が切り替わる。

**キャスティング**: 役職名のみで選ぶ（正体・変化は一切告知しない。完全サプライズ）。

### Bの来歴

1. 過去に施設へ擬態侵入 → 特性ゆえ捕獲・封印（容器に収容）
2. 先遣隊の侵入で封印が緩む → 容器から脱出 → 先遣隊全員を覗く
3. **覗きを通じて先遣隊の高度な知性・外部知識を獲得**。**奪った求心力で隊員の姿を模倣**するが、器は長く持たず灰化 → 次々と乗り換え → 先遣隊は全員崩壊（覗かれただけの隊員＝かろうじて人の形だが身元特定不可 / 器として使い切られた隊員＝灰のみ）
4. 先遣隊の通信途絶 → **施設が自動封鎖（外周封鎖発動）** → Bは施設内に閉じ込められる
5. 封鎖中に施設内を探索 → **封印装置の存在・「封印完了＝封鎖解除」の条件を把握**
6. 外界に出ればより多くの人間に触れられる＝**知的飢餓が動機**。だが封鎖中は脱出不可能
7. **自発的に封印容器に戻る**（いつでも出られる状態で待機）── 次のチームの到着を待つ
8. 第二次チーム到着を検知 → 容器から出る → Phase 3でジャーナリストに**オンタイム乗り換え**

### Bの模倣能力（2026-06-19 追加）

- 覗きで求心力を吸収した後、**奪った力を元に対象の姿を模倣できる**
- ただし奪った器は長く持たず**灰化が進行**するため、定期的に乗り換えが必要
- 先遣隊での行動パターン: 覗く → 姿を模倣 → 器が灰化 → 次の隊員を覗く → 繰り返し
- 現在（ジャーナリストに乗り換え後）は**器ごと入れ替わった状態**。模倣ではなく「ジャーナリストの身体そのもの」を使っている
- ジャーナリストの器もいずれ灰化する＝**時限ジレンマ**（脱出を急ぐ理由）

### Bが乗り換え後に得る記憶・知識

| 知っていること | 知らないこと |
|-------------|------------|
| 自分は上位存在である | **Aが誰に擬態しているか** |
| 過去にこの施設に囚われ封印されていた | 自動認証扉の開錠条件 |
| 先遣隊を覗いて崩壊させた＋その知識を吸収した | 封印の正確な手順（理論面は教授の研究知識が必要） |
| **もう1体の同種がこの場にいる**（求心力の知覚） | |

| **封印完了＝施設封鎖解除**（施設の文書・設備から把握） | |

### 求心力の知覚（民俗学ベース）

Bが覚醒した瞬間、器を通じた「見え方」が変わる。人間には求心力（魂の重み）があるが、**1人だけ自分と同じように空洞の存在がいる**。ただし視界が不安定なため**誰かまでは特定できない**。

- 追加HOに記載:「この中にもう1体、自分と同じ存在がいると感じる。誰かは分からない」
- **Aにも同じ知覚がある**（対称設計）
- 「異形は同類を感じ取る」民俗学モチーフに準拠

## 勝利条件（2段階）

| 段階 | 条件 |
|------|------|
| **初期**（Phase 0〜3前半） | ①施設の実態を取材し記録する ②先遣隊に何が起きたか真相に迫る ③脱出手段を見つける |
| **乗り換え後**（Phase 3後半〜） | ①**正体を秘匿する** ②**施設から脱出する**（＝自分以外の誰かを封印させる） |

### 初期目標の設計意図

人間フェーズでの個人的動機＝**真実を暴くジャーナリストの本能**。科学記事の執筆経験で研究資料を読み解く素地がある。人に聞き情報を引き出す取材行動が、Bの情報操作と二重写しになる。

### 脱出メカニクス（全員共通）

| 手段 | 条件 | Bにとっての意味 |
|------|------|---------------|
| **封印** | 誰かを最奥の封印装置で封印（意識抽出→廃人化）→ ロック解除 | **自分以外の誰かを封印させれば脱出できる**。Aでも人間でもよい |
| **救助待ち** | 封印しない／できない場合 → いずれ救助隊が来る（時間不明） | 脱出は可能だが、時間経過で器が劣化＋バレるリスク |

### 封印装置の仕様

- **依代ごと意識を抽出・格納する装置**（頭に嵌める電極デバイス）
- 上位存在専用ではない。**人間に使用しても同様に意識が抜かれ廃人化する**
- 民俗学的な「魂抜き」を科学で再現した装置
- 覗き（求心力喪失→身体崩壊）とは別の機序：封印装置は意識を抽出するため**身体は残るが廃人**

→ Bの最適戦略は**「自分以外の誰かに嫌疑を向けて封印させる」**。Aを売るのが最善だが、人間を売っても脱出は成立する（ただし本当の脅威＝Bが外に出る最悪の結末）。

## 固有スキル

### 固有スキル（キャラシートに記載・他PLにも見える）
**「査読」** — 研究資料カードから追加情報を読み解く。教授と同じ査読能力だが**固有ターン限定**（最大2回）。査読結果は**口頭報告のみ**（元のカードや原本は他PLに見せられない）。科学記事のファクトチェック経験で自然に見える。

### 処理手順（GM個別ヒアリング）

```
GM → ジャーナリストを個別に呼ぶ（固有ターンは全員順番に個別対応）
  → 【査読】研究資料カードを持っていれば査読。GMが原本版（墨消し解除）を口頭で伝える
  → 戻す
```

他PLには何をしたか見えない。聞かれたら「研究資料を査読した」で通る（科学記事の執筆で論文を読み慣れている）。

### テーマとの接続

- **査読**＝Bが封鎖中に施設を探索し知識を吸収した経験の延長。表の能力として自然に見える（科学記事のファクトチェック経験）

## Phase別

| Phase | 本人の主観・固有イベント | 推奨行動 | ジレンマ |
|-------|------------------------|---------|---------|
| **0. 導入** | 他のメンバーと共に施設に降りる。崩壊した先遣隊を目撃 | 状況把握。取材者として観察・記録を始める | なし（純粋な人間） |
| **1. 第1調査** | **単独調査**。先遣隊に何が起きたのか、独自に情報を集めたい | 手がかり収集。証拠カードを取得し取材者の視点で解釈 | なし（まだ人間。取材者の使命感は本物） |
| **2. 第1会議** | 「人知を超えた存在がいる」と全体が認識 | 情報共有に参加。取材経験に基づく質問力で他者から情報を引き出す | なし（まだ人間） |
| **3. 固有ターン1** | **Bがオンタイムで乗り換え**。追加HOで目標全面切替＋求心力の知覚＋**情報パッケージ**（認証扉の空洞検知機構＋脱走個体の記録）。**査読1回目** | 査読で研究資料の深層情報を取得。**4人目はもう一人の同種だと考えるのが自然** | Phase 0〜2の「普通の自分」と矛盾する言動を取れない |
| **4. 第2調査** | Bとして行動開始。崩壊度の差（装備差）が発見される可能性。自動認証扉に触れると偶然開く可能性 | 脱出に繋がる情報を探す。Aの正体を推理。疑われないよう調査に協力する振り | 自動認証扉＝暴露リスク。崩壊度の差について詳しすぎる反応をすると怪しい |
| **5. 固有ターン2** | **査読2回目** | 査読で後半の研究資料を深掘り | 何を査読したか説明を求められる可能性 |
| **6. 最終調査** | 脱出とA特定を同時進行。Phase 3前後の言動変化が痕跡として浮上する可能性 | Aを特定し封印対象として根拠を集める。自分への疑惑を逸らす | 積極的に動く＝疑われる。消極的＝脱出が遠のく |
| **7. 最終会議＋投票** | **全員投票で封印対象を決定**。弁明の武器＝取材者としての中立性＋Phase 0〜2の「人間だった実績」 | 自分以外に票を集める。民俗学者＝「4人目、施設に詳しすぎる」 | Aが反論してくる。中立性を盾にしつつ積極的に動く矛盾 |
| **8. エンディング** | 投票結果で分岐。全員で対象を最奥の封印装置まで連れて行く | — | — |

### エンディング分岐（B視点）

| 結果 | Bの勝敗 | 描写 |
|------|--------|------|
| **B以外が封印される** | **勝利** | Bは人間のふりをして脱出。知的飢餓を抱えたまま外界へ |
| **Bが封印される** | **敗北** | 意識を装置に抽出される。ジャーナリストの身体は廃人化 |

| **バッドエンド（封印装置に言及ゼロ）** | **全員敗北** | 脱出方法を見つけられず施設に閉じ込められる |

## 比重チェック

- **Phase 0〜2**: 取材者として独自に情報を集める。この間の行動が乗り換え後の偽装基盤
- **Phase 3**: 最重。オンタイム乗り換え＋目標切替＋求心力知覚＋査読1回目
- **Phase 4〜7**: 「嫌疑誘導＋脱出」の暗闘。崩壊度の差の発見が転機
- **Phase 8**: 投票＝最終勝負。取材者の中立性を武器にした弁明 vs Aからの告発

旧PC4（大学院生）からの変更理由:
- **プレイヤー選択の魅力**: 4人の中で最も高度なプレイが求められるポジション（B＝最難関）を、最も魅力的なキャラクターにすべき
- **行動差の隠蔽性**: ジャーナリストは元から「人に聞く・情報を引き出す」が基本行動。乗り換え前後で行動パターンが変わりにくく、PLの隠蔽負担が軽減
- **出歩きの自然さ**: 独自取材のために一人で施設を探索する動機が非常に自然
- **独立性**: 教授との師弟関係による上下関係が解消され、4人全員が対等な専門家として参加する構造に
