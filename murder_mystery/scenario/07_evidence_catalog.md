## F-02. 自動認証扉（固有マス札）

| 項目 | 内容 |
|------|------|
| 枚数 | 1枚（ただし2回のフェーズで訪問可能） |
| 反応条件 | 上位存在（A・B）が接近すると自動開放。人間は無反応（失敗演出なし・沈黙） |
| 奥の証拠 | P4: 論文② ／ P6: 論文④ |
| 裏面デザイン | 注連縄鳥居＋走査装置（`final_back_biodoor`） |
| 設計意図 | **開けた事実自体が証拠**。情報 vs 疑惑のジレンマ |

### 設定: 漏斗型誘導設計（2026-06-18 確定）

扉の設計思想は「**一方向トラップ**」。

- **本来の動作**: 外周 → 最奥区画方向にのみ開放。通過後に閉鎖し、上位存在を封印装置へ段階的に追い込む
- **現状**: **経年劣化により閉鎖機能が死んでいる**。検知・開放は動作するが、通過後に閉じない。施設全体が「開くだけ開いて閉じない」状態
- **二層構造との関係**: 論文④§1で説明する「外周封鎖」（施設の出入口遮断）と「自動認証扉」（内部区画間制御）は別系統。外周封鎖は正常稼働中、自動認証扉の閉鎖側が壊れている

### カードのフレーバー表現

> **表現方針**: 高度な技術が使われているのが分かるが、動いているのが不思議なほど老朽化している

- **視覚**: 精密な走査機構の痕跡（電極ノード、配線）＋ 錆・亀裂・苔・剥落。光る部分（走査線）が辛うじて点滅している
- **テキスト案**:
  - 「区画間に設置された認証機構。精緻な走査装置の痕跡が見て取れるが、外装は著しく劣化している。それでも、近づく者があれば──まだ、反応する。」
  - （短縮版）「老朽化した走査装置。それでもまだ、反応する。」

## P1証拠カードの設計方針（2026-06-18 メモ）

> P1（8枚）は「**この施設は何なのか、何がいるのか**」を掴むフェーズ。以下のテーマで組む:
>
> - 施設の設立経緯（上位種の発見→研究施設の建設）
> - 上位種の捕獲・収容の記録断片
> - 先遣隊の活動形跡（装備、通信記録、日誌断片など）
> - 「人ならざるものを研究していた」ことの物証
> - → 全体で「**化け物がいる施設だ**」をプレイヤーに叩き込む
>
> **査読ログ（OBS-0041）の再考**: 表面テキストはP1の空気に合わせた基礎的な観察記録（上位種の存在証拠）寄りに。査読で開く裏面に「2系統の反応パターン」のヒントを仕込む形で、種カード機能を維持。
