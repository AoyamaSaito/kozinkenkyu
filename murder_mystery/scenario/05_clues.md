## 設計原則

1. **特定は「物的証拠の消去法格子」で組む**。各物証が4人を集合に分割し、**プレイヤー自身が消去法で交点（＝ジャーナリスト＝B）に収束**させる。推理ゲームの土台は「**疑わしい人を増やす仕組み**」＋「**最後に消去法で1人に絞る仕組み**」。
2. **主筋の3点**：①認証扉テスト（誰が上位存在か）／②崩壊痕跡＋研究（入れ替わった者がいる）／③出歩き（誰が入れ替わりの隙を持つか）。
3. **動的証拠（覗き被害・崩壊カウンタ）は補強・緊張に格下げ**（§C）。
4. **魂抜き検査は保険**（§E）。扉テストが外れた卓向けの難易度調整用。**一旦不採用**。
5. **最終確認**は封印実行（正しければロック解除＝物理的な自白代替）。

## A. 消去法格子（特定の本筋）

### 物証1 ── 認証扉テスト（2回運用）：**誰が上位存在か**を割る【本体】

**認証の由来（設定から自然に降りる／無理やりでない）**：施設は**上位存在を捕獲・研究していた** → 異形を検知する認証を備えているのが当然。認証は「**求心力を持つ者（人間）を通し、空洞の者（異形）に反応する**」＝施設の対異形セキュリティ。**身体ではなく中身（空洞か）を読む**ので、**Bがジャーナリストの身体を着ていても反応する**。

| 項目 | 内容 |
|---|---|
| 割る集合 | **{民俗学者・ジャーナリスト＝上位存在(空洞)}** ／ {調査隊員・教授＝人間(求心力あり)} |
| 物 | 固有マス「自動認証扉」カード（固有裏 `back_biodoor`）。奥に研究資料＝開ける動機 |
| 反応仕様 | その場のペアに**上位存在が1人でもいれば開く**。ただし**どちらが反応したかは分からない**。**開かない＝2人とも人間（白）** |
| 運用 | **2回**（探索構造の帰結＝扉マスは各探索フェーズで1回 × P4・P7の2フェーズ）。**扉は基本ランダム要素**——通常のバディ調査で誰がそのマスを訪れるか次第で反応する（狙った"送り込み"運用は作り込まない／詰めは後で） |
| 意味の開示 | **P7のカードで「扉＝空洞を弾く異形検知」と判明**（→ それ以前の反応が**伏線として回収**＝「あの時開いたのは人間じゃないからだ」）。カード前でも開く/開かないのパターンから経験的に推測は可能 |
| 推理要素 | 「**開かない＝2人白**」を引ければ残り2人が上位存在に確定。**誰を扉マスに連れて行くか**（通常のバディ選択）＋出歩き情報の突き合わせ |
| 注意 | ランダム性ゆえ**情報が出ない卓もありうる**（証拠になる時とならない時がある）。その時の保険が§E。**主軸は出歩き＋論理**が支える |

> **2回でBに届く例**：1回目(ジャーナリスト・調査員)→開く→「どちらかが上位存在」／2回目(調査員・教授)→開かない→**調査員・教授とも白**→∴**ジャーナリスト＝上位存在**。「開かない＝2人白」を1回引けば残り2人が上位存在に確定するのが効く。

### 物証2 ── 先遣隊の崩壊痕跡＋研究資料：**「入れ替わった者がいる」**を割る
| 項目 | 内容 |
|---|---|
| 割る集合 | （人を名指さない）「紛れ込みが怪しい」ではなく「**そもそも器を乗っ取って入れ替わった者がいる**」を研究資料と合わせて築く |
| 物 | Phase4現場：灰＋装備の残骸／**崩壊度の差**（1体だけ"ほぼ灰"＝器として使い切られた＝Bの前の器）＋研究資料（乗っ取りのMO） |
| フェーズ | Phase4 |
| 役割 | 「**入れ替わりには単独の隙が要る**」の土台 → 出歩き（物証3）に接続。**A（余計な1人）とB（入れ替わった1人）は別人**という二重構造を立てる |

### 物証3 ── 出歩き（アリバイ）：**ジャーナリスト・調査員**に容疑を集める
| 項目 | 内容 |
|---|---|
| 割る集合 | 容疑を上げる：**{ジャーナリスト・調査員＝単独で出歩いていた2人}** |
| 物 | 固有ターンの目標更新と**セットで開示**する目撃情報。**民俗学者と教授に、出歩いていた2人を1人ずつ**目撃させる |
| フェーズ | **固有ターン（目標更新時）に開示＝扉の使用より前**。よって扉の反応を解釈する材料が先に揃う |
| 理由(in-fiction) | ジャーナリスト＝**独自取材のため施設内を一人で探索・撮影していた**（取材者の行動として自然）／調査員＝通信復旧のため直近設備を再点検 |
| 駆け引き | 目撃は**1人1証言**＝両者を確定するには2人が証言を突き合わせる必要。調査員は理由を**証明できない**＝疑いが残る |
| 役割 | 入れ替わりは**単独行動の隙**に起きた（出歩き＝乗っ取りの窓）。出歩いた2人の**どちらが上位存在かは扉テストで割る**（ジャーナリスト＝反応／調査員＝無反応） |

### 交点 ＝ B特定（魂抜き検査なしで届く）
1. **扉テスト** → 上位存在2人を割る（民俗学者・ジャーナリスト）。「開かないペア」で人間も白にできる
2. **崩壊痕跡＋研究** → 「入れ替わった者がいる（隙が要る）」
3. **出歩き** → 隙を持っていたのはジャーナリスト・調査員
4. **論理**：上位存在のうち、**出歩いて隙のあった方＝入れ替わったB＝ジャーナリスト**。民俗学者は上位存在だが出歩いてない＝入れ替わってない＝**余計な1人A**。調査員は出歩いたが扉が反応しない＝**人間**
- → **ジャーナリスト＝Bが一点に落ちる**（運ゲー化しない）

## B. ソフトな罠（誤誘導 ── 推理の山）

**赤いニシンが3枚**（濡れ衣濃度＝Aが最濃／人間2人は中。→ `03_info_layers §濡れ衣材料の総量`）。どれもハード物証ではなく**疑念**で、素直に追うと無実の人物を誤封印する：

- **罠①（最濃）：紛れ込み → 民俗学者(A)**。「本来3名・4人目は想定外」＋施設に詳しすぎる＋過去の逃走記録＋扉が反応。しかも**既存設定だけで割れる**——調査員＋教授＋ジャーナリスト＝正規3名 → 余る民俗学者が一発で"侵入者"に。**この割れやすさは撒き餌**。※ただし**断定はさせず言い逃れの余地を残す**——民俗学者の弁明（「部外者だが誰も傷つけていない」）が**実は真実**＝そこが逃げ場
- **罠②（中）：出歩き → 調査員**。アリバイを証明できず怪しいが、人間（無実）。証拠に最も触れる＝「なぜそんなに知っている？」も上乗せ
- **罠③（中）：引用元疑惑 → 教授**。自分の研究が施設に転用された関係者では？＝白なのに疑われる（査読で情報を出すほど疑惑が増す）

**なぜ危険か**：
- B（ジャーナリスト）は「**侵入者＝民俗学者を封じろ**」と全力で誘導する
- 民俗学者(A)も上位存在＝**封じればロック解除で脱出成立＝Bの勝ち**
- → 罠にハマると**Bが勝つ**。緊張源として強力

**命綱＝扉テストの2回を活かすこと**＋「入れ替わりには隙が要る」論理。これが罠から抜ける道。使いこなせなければ騙されてBが勝つ（＝難易度）。

**反転の論理**：枠が異常（余計な1人）＝民俗学者(A・無害)／中身が異常（魂を抜かれた器）＝ジャーナリスト(B・危険)。「誰がいるべきでないか」でなく「**誰の身体が乗っ取られてるか**」を問えばジャーナリストに落ちる（→ `02_beatsheet.md §二重構造`）。

## E. 保険（一旦不採用）── 魂抜き検査

扉テストが外れた卓向けの **backup**。難易度調整で戻す場合の設計メモ。

- 根拠（成熟モデル）：Bは自前の体を作れず人を殺して器を奪う → **ジャーナリストの器は魂抜きの犠牲者＝痕跡を持つ**。Aは自前の体＝痕跡なし
- 構成：(3a)研究資料から「魂抜き被害者の症状プロファイル」を入手 → (3b)4人に照合し、痕跡を持つ者（＝ジャーナリスト）を判定
- 整合：検査が読むのは「この器に**魂抜きが起きた痕跡**があるか」（B占有で機能していても抽出の痕跡は残る前提＝廃人化とは別）
- 戻す時の判断点：(3b)の照合手段を①誰でも可能な調査／②封印装置に紐付け／③特定スキル のどれにするか
