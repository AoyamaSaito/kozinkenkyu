# 09a. P1 証拠カード ── 初期調査フェーズ（8枚）

> **位置づけ**: P1（第1調査フェーズ）で探索マスから取得する証拠カード8枚の本文。
> `design_card.py` の `make_evidence_card_wa(variant="ofuda")` で画像化する。
>
> **カード仕様**: 600×840px / font23 / 18文字折返し / 行高36 / header + prefix + No.
>
> **P1全体のゴール**: 「この施設は人ならざるものを研究・収容していた場所だ」をプレイヤーに掴ませる
>
> **関連**: `07_evidence_catalog.md`（証拠品カタログ）/ `05_clues.md`（情報配置）

## P1-01. 施設銘板（破損）

| 項目 | 値 |
|------|-----|
| header | 物的証拠 |
| prefix | C |
| No. | 01 |
| title | 施設銘板（破損） |
| 分類 | 雰囲気 |

### カード本文

```
入口付近の金属銘板。
「○○県御影村
　██████研究所」
名称の一部が
意図的に削り取られている。
裏面に設立年の刻印のみ。
```

### 設計意図

- 「御影村」の地名は民俗学的な含みを持つが、P1ではただの地名として流れる
- 名称が削られている＝この施設は存在を隠したかった
- 情報量は少ないが「ここは公にできない場所」という第一印象を作る

## P1-02. 先遣隊 活動記録

| 項目 | 値 |
|------|-----|
| header | 記録 |
| prefix | L |
| No. | 02 |
| title | 先遣隊 活動記録 |
| 分類 | 情報（日常→途絶） |

### カード本文

```
[██年5月12日]
第3日。B区画の調査を継続。
設備の劣化が激しいが
通行に支障なし。
明日は奥の区画に入る予定。

──以降の記録はない。
```

### 設計意図

- 何てことない活動報告が突然途切れている不気味さ
- 「奥の区画に入る予定」＝奥に進んだ結果どうなったかはPhase 0で目撃済み（崩壊した先遣隊）
- **日付がP1-06（収容監視ログ）の異常検知日と一致する**。プレイヤーが2枚を突き合わせて気づく設計

> **GM注**: P1-02とP1-06の日付は「██年5月12日」で統一済み。年号のみGMが設定する。

## P1-03. 防護服のロッカー

| 項目 | 値 |
|------|-----|
| header | 物的証拠 |
| prefix | C |
| No. | 03 |
| title | 防護服のロッカー |
| 分類 | 雰囲気 |

### カード本文

```
壁面のロッカーに
特殊な防護服が数着。
内側に焦げた跡のあるもの、
未開封のまま残されたもの。
通常とは異なる構造をしている。
何を想定した装備なのか。
```

### 設計意図

- 「焦げ跡」＝覗きの崩壊（身体が灰化する）を間接的に示唆するが、P1時点では繋がらない
- 「何を想定した装備なのか」は読み手への問いかけ。答えはP4以降で分かる
- 未開封品がある＝施設放棄が急だったことを示す

## P1-04. 古い神棚

| 項目 | 値 |
|------|-----|
| header | 物的証拠 |
| prefix | C |
| No. | 04 |
| title | 古い神棚 |
| 分類 | 雰囲気 |

### カード本文

```
研究室の壁に設えられた神棚。
注連縄と御札。
「鎮め」と墨書された木札が
一枚だけ床に落ちている。
科学施設には似つかわしくない。
```

### 設計意図

- 「科学×民俗」の融合を最初に見せるカード。シナリオ全体のモチーフを提示
- 「鎮め」＝封印のヒント（封印装置の詳細はP6論文④で判明）
- 木札が「落ちている」＝鎮めが機能していない暗示

## P1-05. 研究日誌の断片

| 項目 | 値 |
|------|-----|
| header | 記録 |
| prefix | L |
| No. | 05 |
| title | 研究日誌の断片 |
| 分類 | 情報（異常な研究対象） |

### カード本文

```
対象の観察を継続する。
既知の生物学的分類に該当しない。
接触時の反応パターンを
引き続き記録する。
差し当たり、危険性は低い。
```

### 設計意図

- 「上位存在を研究していた」ことの最も直接的な証拠
- 淡々とした研究者のトーンで書かれているが、内容は異常（既知の分類に該当しない）
- 「危険性は低い」＝この認識が後に覆される伏線。施設が崩壊した現状との落差

## P1-06. 収容監視ログ

| 項目 | 値 |
|------|-----|
| header | 記録 |
| prefix | L |
| No. | 06 |
| title | 収容監視ログ |
| 分類 | 情報（開放の事実＋日付） |

### カード本文

```
[自動記録]
██/██ 収容状態：正常
██/██ 収容状態：正常
██/██ 収容状態：正常
██/██ 収容状態：正常
██/██ 収容状態：異常
──Loss detected──
以降の記録なし。
```

### 設計意図

- 淡々とした自動記録の羅列。「何が」出たかは一切書かない
- 「Loss detected」＝英語の機械出力が混じることで自動システム感を出す
- **最終行の日付がP1-02（先遣隊の活動記録）と一致する**。2枚を突き合わせると「先遣隊が来た日に何かが出た」と分かる
- 「以降の記録なし」＝監視システム自体が止まった

> **GM注**: P1-02とP1-06の日付は「██年5月12日」で統一済み。異常検知日＝先遣隊が奥の区画に進んだ日。

## P1-07. 特殊保管容器（空）

| 項目 | 値 |
|------|-----|
| header | 物的証拠 |
| prefix | C |
| No. | 07 |
| title | 特殊保管容器（空） |
| 分類 | 雰囲気 |

### カード本文

```
頑丈な作りの容器。
内壁に緩衝材と固定具。
明らかに何かを
収めるための形状だが
中は空。
蓋の内側に掻き傷がある。
```

### 設計意図

- 「何が入っていたのか」は語らない。容器の異質さだけで「ここでは何かを閉じ込めていた」を示す
- 「掻き傷」＝中から出ようとした痕。直接的に言わないが不穏さを残す
- P1-06（収容監視ログ）と組み合わさると「中身が出た」と推測できる

## P1-08. 査読ログ OBS-0041

| 項目 | 値 |
|------|-----|
| header | 観察記録 |
| prefix | OBS |
| No. | 41 |
| title | 定期観察記録 |
| 分類 | 情報（査読で二経路の種） |
| 査読 | 可（教授のスキルで追加情報開示） |

### カード本文（公開面）

```
管理番号: OBS-0041
[██年5月12日]
定期観察記録。被験者7名中、
5名に認知機能低下を確認。
2名は異常なし。記録と一致せず。
2系統の反応パターンを観測。
要精査。　　　[署名判読不能]
```

### 査読で追加される情報（教授のみ）

> **実装**: 査読後にGMが口頭または別カードで開示。

```
「2系統」──すなわち、
被験体の認知構造が
保持される場合と、
不可逆的に崩壊する場合がある。
前者の発生条件は未解明だが、
後者は求心力の完全喪失と一致する。

両経路を分岐させる条件の解明は
本施設の最優先課題である。

→ 内部紀要 Vol.7 にて理論化を予定
```

### 設計意図

- **段階Aの最低保証**。教授がP3固有ターンで査読すれば二経路の取っかかりを得る
- 公開面だけでも「覗き込み」「認知機能の低下」「2系統」というキーワードが出る
- 「内部紀要 Vol.7」＝P4以降の論文①②への導線
