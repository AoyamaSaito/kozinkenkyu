## P6-01. 逃走記録

| 項目 | 値 |
|------|-----|
| header | 記録 |
| prefix | L |
| No. | 15 |
| title | 逃走記録 |
| 分類 | 情報（罠①強化→反転材料） |

### カード本文

```
[施設保安記録]

██/██　収容区画より逸脱1件。
追跡チームを展開。

確保に至らず。
捜索を打ち切る。
──対象の所在は現在も不明。
```

### 設計意図

- **罠①（民俗学者への嫌疑）の最終強化**: 「過去に何かが逃げた」→「4人目（想定外のメンバー）＝逃げた奴＝民俗学者？」とPLが推理する誘導
- 個体αβの名前は出さない。論文③（成熟モデル）と突き合わせて「脱走個体＝成熟済＝共鳴＝善」と分かったときに反転が起きる
- **「所在は現在も不明」** ＝逃げた存在がまだどこかにいる。施設外か、あるいは——この中に？
- 「逃げた奴＝危険」は人間の自然な推論。だが実際は「逃げた方が善」で「残された方が悪」＝直感の裏切り

## P6-02. 封印装置

| 項目 | 値 |
|------|-----|
| header | 物的証拠 |
| prefix | C |
| No. | 16 |
| title | 封印装置 |
| 分類 | 雰囲気（エンディング予告） |

### カード本文

```
最奥区画に据えられた装置。
椅子と、頭部に嵌める電極。
壁面の銘板に一行──
「意識の抽出および格納」
注連縄が巻かれている。
まだ使える状態に見える。
```

### 設計意図

- Phase 8（投票→封印）で起きることの物理的予告。「これに座らされる」恐怖
- 「意識の抽出および格納」＝封印装置が何をするか（意識を抜く＝廃人化）を端的に伝える
- 「注連縄」＝P1-04（神棚）/ P4-01（拘束具）から続く科学×民俗の融合。施設の最奥に至ってもこの二重性は健在
- 論文④（封印手順）が「使い方」を提供。このカードは装置の存在と恐ろしさだけを伝え、手順は論文に委ねる

> **※設計課題**: 「封印＝脱出」の接続が証拠品依存になっている問題あり（別途検討中）
