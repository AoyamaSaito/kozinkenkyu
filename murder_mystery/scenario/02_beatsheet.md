## 固有ターン詳細

### 手順

```
GM「固有ターンを開始します。1人ずつお呼びします」
  → 各PLを順番に個別対応（別室／個別通話）
    → キャラシート更新の案内
    → スキル持ち: どこに／誰に使うか宣言 → 結果を受け取る
    → スキルなし: 更新案内＋質問対応のみ
    → 「施設の情報で目標に変化があります。質問があれば時間を取ります」
  → 全員完了で次のPhaseへ
```

> **運用**: 他PLには誰が何をしたか見えない。全員に同じ時間枠を取ることで特定PLの行動が目立たない。

### 固有ターン1（Phase 3）

「人知を超えた存在がこの中にいる」と認識した直後。**全員のキャラシートが更新**される。
A・Bはここで**自覚**（＝追加の秘匿情報を受け取り、自分が人間でないことに気づく）。

| キャラ          | 行動                         | キャラシート更新内容                          |
| ------------ | -------------------------- | ----------------------------------- |
| **民俗学者**（善A） | **想起**が自動発火 → 封印装置の場所を思い出す | **自覚**：自分は人間ではない。新目的：Bを封じ、自分は生存する。**4人目＝もう一人の同種の可能性**   |
| **調査隊員**（人間） | スキルなし（開錠は探索中に使用）           | 目的改訂：崩壊の原因は人外の存在。**4人目の正体を特定し確保する**（初期任務「施設関係者の確保」と接続）       |
| **大学教授**（人間） | 研究資料を持っていれば**査読**1回        | 目的改訂：自分の研究が関わっている可能性。理論面の追及。**4人目の正体を特定する**         |
| **科学ジャーナリスト**（悪B） | **査読**1回＋**情報パッケージ**（認証扉の空洞検知＋脱走個体の記録）    | **自覚**＋**求心力の知覚**（もう1体いるが誰かは不明）。新目的：正体を隠し、自分以外を封印させて脱出する。**4人目＝もう一人の同種だと考えるのが自然** |

### 固有ターン2（Phase 6）

第2調査の後。証拠が出揃い始め、後半戦（最終調査→決着）に入る前の**最後の個別行動機会**。

| キャラ | 行動 | 備考 |
|--------|------|------|
| **民俗学者**（善A） | **気配感知**2回目が自動発火。GMが「調査隊の誰かが入れ替わっている。一夜目を境に気配が変わった」を通知 | 出歩き情報と組み合わせてBの特定材料に使える |
| **調査隊員**（人間） | スキルなし（開錠は探索中に使用） | 更新案内のみ |
| **大学教授**（人間） | 研究資料を持っていれば**査読**1回（2回目） | 後半の情報深掘り |
| **科学ジャーナリスト**（悪B） | **査読**1回（2回目） | 後半の研究資料を深掘り |

> **比重の注意**: 固有ターン2で行動があるのは民俗学者（気配感知2回目）・教授・ジャーナリスト。調査隊員は固有ターン1と探索フェーズで比重を担保。テストプレイで偏りが出れば調整。

## バディ調査の運用規則（2026-06-30 追加）

Phase 4・7はバディ（2人1組）でGMと調査。**1組ずつ順番にGMが対応**する（並行ではない）。

### 待機ペアの行動
GMが調査中の1組を対応している間、**待機ペアは密談（自由な情報交換・推理の相談）が可能**。ゲーム上の公式行動として位置づける。

### 指名順ローテーション
バディの指名順をPhase間でローテーションし、**同じプレイヤーが連続で後回しにならない**ようにする。

| Phase | 先攻指名 | 後攻指名 |

| **7** | Phase 4で後攻だった組が先攻 | Phase 4で先攻だった組が後攻 |

## 設計メモ（要・後続検討）

### ★ Phase 8 最終会議＋投票（2026-06-18 更新）

**4段階の内部フロー:**

```
① 脱出手段の提示
  GM（調査本部からの通信）:「こちら調査本部。外周封鎖の解除には、承認済みプロトコルの実行が必要だ。
  　脱出手段を提示してください。」
  → ラウンドロビン（1人ずつ脱出案を述べる）
  → 4人中1人でも「封印装置を使う」方向の言及があれば合意成立
  → 言及ゼロ＝バッドエンド（脱出方法を見つけられず終了）

② 推理発表
  → 1人ずつ「誰を封印すべきか」＋根拠を述べる

③ 自由議論
  → 反論・弁明・最終弁論

④ 投票
  → 最多票の人物を全員で最奥の封印装置まで連れて行き封印
```

- 封印装置は**誰にでも使える**（意識抽出→廃人化）。人間を誤封印しても脱出は成立するが無実の犠牲
- 拮抗の源泉: **Bが自分以外に票を集める情報操作** vs **A＋人間がBを正しく特定する推理**
- **AもBも封印装置を知っている**（A=HO想起、B=新タイムライン）。両者とも言及する動機がある＝バッドエンドは構造的にほぼ発生しない。ただし「先に言った奴が怪しい」の駆け引きが生まれる
- 同票時の処理は未確定（決選投票 or GM裁定）

### ★ Phase 9 Aの処遇（2026-06-30 追加・同日改訂）

メイン封印の**後**、調査隊員が生存している場合に発生するサブエンディング。調査隊員の初期任務「施設関係者の特定・確保」の回収。

**発生条件**: **調査隊員が封印されていない**こと。（調査隊員が封印された場合はスキップ）

**処理フロー**:
1. メイン封印＋結果描写が完了
2. GMが調査隊員に個別質問:「施設関係者について——この中に関係者だと思われる人物はいますか？」
3. 調査隊員が**誰かを指名する**か、**誰も指名しない**かを判断
4. 指名した場合、**確保するかしないか**を調査隊員が判断

**4パターン分岐**:

| パターン | 調査隊員の判断 | 結果 |
|---------|--------------|------|
| **A) 正しく特定＋確保** | 民俗学者を施設関係者と特定し、確保する | Aは身柄を拘束され連行される。調査隊員の任務完遂 |
| **B) 正しく特定＋解放** | 民俗学者を特定するが、無害と判断して解放 | Aは自由の身。調査隊員は任務を果たさない選択をした |
| **C) 誤認特定＋確保** | 別の人物（教授など）を施設関係者と誤認し確保 | 無実の人物が拘束される。Aは見逃される |
| **D) 誰も特定しない** | 「分からない」「いない」と判断 | 処遇はスキップ。Aは判明しないまま脱出 |

**設計意図**:
- Aが安易にカミングアウトすることへの抑止力。テストプレイで「Aが自分から正体を明かして味方に付く」が発生したため、**正体開示＝確保リスク**のジレンマを追加
- 「悪を封じる」だけでなく「善をどう扱うか」という倫理的判断が残る
- 調査隊員の「確保」目標が最終的にここで回収される
- **判定は調査隊員の単独判断**。議論ではなくGMからの個別質問で処理し、エンディングの尺を抑える

### 「紛れ込み」と「なりかわり」の二重構造（Phase 3〜4）
PLが解くべき謎は**2つの異常が同時に存在する**こと：

| 謎 | 発覚Phase | 事実 | PLの見え方 |
|----|----------|------|-----------|
| **① 人数不一致** | Phase 3（GM通信） | 本来の調査メンバーは3名。**善玉Aが身分を偽って潜入**した結果4人になっている | 「誰かが紛れ込んだ」 |
| **② 崩壊痕跡の発見** | Phase 4（第2調査） | **悪玉Bが正規メンバー1人になりかわった**。元の人間は求心力を失い崩壊（灰＋装備の残骸） | 「誰かが入れ替わっている」 |

- **余計な1人**（＝A）と**入れ替わった1人**（＝B）は**別の人物**。PLはこれを区別しなければならない
- Bは「紛れ込んだ奴が怪しい」と誘導できる（Aに濡れ衣）
- Aは「入れ替わった奴こそ危険」と主張できるが、自分が「紛れ込んだ側」であることを隠す必要がある
- **2つの異常を混同させるか・区別させるかが推理の核心**

### 深層のほころび（Phase 7・見分けの手がかり）
- **善玉A**：人としての過去が薄い・空白（擬態だが人格は保つ → 記憶の地に足がつかない）
- **悪玉B**：成り代わった元人間の記憶と、本性がチグハグ（器を壊しつつ被るので齟齬が出る）

### 固有ターンの比重バランス
- 民俗学者・調査隊員は固有ターン2でスキルなし → **探索フェーズでの行動量**（想起・開錠・自動認証のジレンマ）で補償
- テストプレイで「固有ターン2がスカスカ」と感じたら、民俗学者に2回目のスキル（例：伝承の断片想起）を追加検討
