# 参考構成案: 狂気山脈『陰謀の分水嶺』のスキル設計

> **用途**: 自作シナリオ『覗きの代』のスキル設計の参考資料。
> ゲームメカニクスの設計パターンとして保存。

---

## 基本構造

- 各キャラに**1〜2個の離散的アクション**（固有スキル）
- 全て「**いつ・何回・何が起きるか**」が明確
- 共通点: **通常ルールを1箇所だけ破る**（見れないものを見る／取れないものを取る／知れないことを知る）

## キャラ別スキル一覧

| キャラ | スキル名 | 効果 | 使用条件 |
|--------|---------|------|---------|
| 登山家 | 登攀 | ソロでも難所カードを取得可能（通常はバディ限定） | 【登山装備】カード3枚所持時。複数回使用可 |
| 記者 | 言いくるめ | 相手の非公開カード1枚を**強制閲覧** | 密談時のみ |
| 調査員 | 忍び歩き | 相手のカード1枚を**指定して奪取** | 密談時のみ |
| 教授 | クトゥルフ神話 | 場の未取得カード1枚を**覗き見して戻す**（情報偵察） | **1回限定** |
| 医者 | 精神分析 | 相手が【発狂状態】か**判別** | 密談時 |
| 医者 | 電撃発生装置 | 相手を殺傷・気絶（特殊アイテム） | エンディング限定・1回 |

## 調査フェーズの構造

### フェーズ構成
- **前半**: ソロ調査×2回 → バディ調査×2回
- **後半**: ソロ調査×2回 → バディ調査×1回

### ソロ調査
- 盤上のカードを1枚取得
- 標高制限あり（下から順番に取得）

### バディ調査
- 手番PLが相手を1人指名、2人で行動
- カード内容は両者共有、所有権は指名側
- **密談限定スキルはバディ調査時のみ発動可能**
- 同一相手への複数回指名は原則禁止

## 設計パターンの抽出

1. **スキルは「通常ルールの例外」として定義**: 普段できないことが1つだけできる
2. **使用条件で差別化**: 密談限定 / 回数限定 / アイテム前提 / フェーズ限定
3. **情報の流れに介入する**: 見る・奪う・偵察する・判別する——全て情報戦
4. **攻防の非対称**: 記者は「見る」、調査員は「奪う」。似ているが強度が違う
