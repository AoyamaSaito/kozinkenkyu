## §1 設計方針

### §1.1 基本原則

<!-- この作品の情報設計で守るべき原則を列挙 -->

1. **逆漏斗の原則**: 証拠は集まるほど容疑を「外→内」へ狭める。序盤は外部犯説が自然に見え、調査が進むほど「犯人は招かれた4人の中」へ収束する
2. **格子は「無実の消去」に使う**: 通常の犯人特定格子と逆向き。証拠が各PCの「前夜、現場にいた」事実を1人ずつ立証していき、最終的に全マスが埋まる（＝全員当事者の露見が証拠側から保証される）
3. **事件性そのものが誤認**: 「窃盗事件」という前提自体が崩れる設計。段階C以降の証拠は「盗まれたのではない」方向を指す
4. **反転（段階D）は運任せにしない**: 最終議論の前に全員選定の真相が確定的に開示されるよう、決定打を固定イベント化する（詰み防止の最重要ライン）
5. **依頼フェーズは動的証拠の供給源**: 調査フェーズ＝静的証拠（過去の痕跡）、依頼フェーズ＝動的証拠（断片武器の共鳴・戦果異常・暗殺勢力の接触）。両輪で段階を進める
6. **各PCの黙秘を織り込む**: 全員に強い隠匿動機があるため、自白に依存する推理経路を作らない。物証と伝承だけで段階Dまで到達可能にする

### §1.2 推理の段階設計

<!-- プレイヤーの理解がどの順序で深まるべきかを定義 -->

| 段階 | 到達すべき理解 | いつ頃到達するか |
|------|--------------|----------------|
| A | **外は消える**: 外部犯は不可能（儀式の間には素質なき者は入れない等）。犯人は内側＝4人の中にいる | 1巡目（調査→依頼→会議） |
| B | **全員が夜、あの場にいた**: 4人全員の前夜の行動が痕跡から立証される（全員当事者の露見） | 1巡目後半〜2巡目 |
| C | **盗難ではない**: 聖剣は変形する。各自の「見知らぬ武器」＝剣の断片。剣は盗まれたのではなく分かれた | 2巡目 |
| D | **全員で一人の選ばれし者**: 選定の資は四つ（勇気・叡智・信仰・狡智）。選定は前夜、既に成されていた | 2巡目会議後の固定イベントで確定開示 → 終盤特殊依頼・最終議論へ |

> 全体構成（確定）: 導入・HO読み → 1巡（調査→依頼→会議）→ 2巡（同）→ 固定イベント（段階D）→ 終盤特殊依頼 → 最終議論 → 投票/交渉。想定 約3〜3.5h。
> 依頼タッグは全3通り中2通りしか組めない（相方選択＝希少資源）。

> 段階A/Bの間は並行進行を許容する（Bが先に割れても崩壊しない：Aが締まるまで「外部犯に嵌められた」説が残る）。
> C→Dは順序依存（変形の理解なしに全員選定へは飛べない）。

## §5 消去法設計

### §5.1 容疑格子

<!-- 本作の格子は二層構成（§1.1 原則1・2 参照）。証拠の実体は 05_clues.md で設計 -->

**層1: 外部犯消去格子（段階A用）** — NPC容疑者を並べ、証拠で1人ずつ消す通常型の消去法。
最後の1人が消えた瞬間「では誰が？」が内側に向く。

| 証拠 | NPC容疑者1 | NPC容疑者2 | NPC容疑者3 | … | 備考 |
|------|-----------|-----------|-----------|---|------|
| （未設計: NPC容疑者の人数・顔ぶれは未決事項7） | | | | | |

**層2: 割当格子（段階B〜D用・4×4: PC×資質）** — 「誰がどの資質か」を当てる客観推理の器（00_concept.md §E.1）。
旧・当事者立証格子（誰が関与したか）はこの格子の行の開放（在場証明）として吸収。

**解法の二段合成**（設計の芯）:

> **PC→発現形**（依頼の滲み・信仰の目撃・前夜の痕跡） × **発現形→資質**（碑文の辞書） ＝ **PC→資質**

**充填チャネル表**（どの情報源が格子のどこを埋めるか）:

| チャネル | 供給元 | 埋まる場所 | 備考 |
|---------|--------|-----------|------|
| 自分のHO | 秘匿層 | 自分の行（断片の形は既知。ただし資質名は碑文が要る） | 各PLは開始時点で1行の半分を持つ |
| 聖剣スキルの滲み | 依頼フェーズ（相方の観測） | 相方の行×発現形 | タッグ2回＝最大2人分。誰と組むかで埋まる行が決まる |
| 信仰の目撃 | HO秘匿層 | 特定1人の行（在場） | 格子の速度調整弁。開示は信仰PLの決断 |
| 前夜の痕跡（調査4枚） | 調査プール | 各PCの行（在場証明） | 段階Bの底support。1人1枚で速度を均す |
| 碑文（調査5枚） | 調査プール | 列の意味（発現形→資質の辞書） | 段階Cの幹。「狡なき剣は死ぬ」等 |
| NPC消去（調査5枚） | 調査プール | 格子の外周を閉じる（段階A） | 外部犯候補の不可能証明 |
| 依頼情報（4件×2ターン＝8スロット） | 依頼フェーズ成果 | **核心**: 他PCの秘密への接近・資質発現の直接記述・断罪者の動き | 段階制（完全/部分/痕跡）。比重最重 |
| ミスリード（調査2枚） | 調査プール | 解釈揺れ（古語の訳の対立等） | 埋めるのではなく曇らせる |
| 固定イベント | 2巡目会議後 | 格子全体の答え合わせ＋段階D | 「四資質は一つの器に宿らず」 |

**検証すべき性質**（05_clues.md での実装時にチェック）:
1. どのタッグ編成を選んでも、固定イベント前に格子が**8割方**埋まりうる（依頼全失敗の最悪ケースでも痕跡情報＋調査で段階Cまで到達可能）
2. 1人だけ極端に行が開くのが早い/遅い状態を作らない（単独犯疑いでの停滞防止）
3. 聖剣スキルを**使わなかった**PCの行は滲みで埋まらない＝「使わない」が情報の防御として本当に機能する（ジレンマの実効性）

凡例: ◎=強く示唆 / ○=やや示唆 / △=弱い示唆 / ×=否定 / −=無関係
