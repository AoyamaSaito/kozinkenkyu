# 07. ビジュアルデザイン仕様 ── 配布物の共通デザイン言語

> **位置づけ**: 配布物（導入シート・HO・証拠カード・ルール説明等）のデザイン方向性の「正」。
> モック実物: `tools/mockups/intro_sheet.html`（導入シート v1）
> 決定: 2026-07-13（00_DECISIONS.md 記録済）

---

## 1. コンセプト

**「2077のUI文法 × 赤の時代の大気」**

- ベースは Cyberpunk 2077 のUIビジュアル（青黒地・HUDシアン・サイバーイエロー）。プレイヤーが一目で「あの世界だ」と分かる既視感を利用する
- そこに RED（2045年）の固有要素として**赤いもや**を大気として流す。〈赤の時代〉＝赤い空の時代性をデザインで語る
- 前作『覗きの代』（明朝・和ホラー・金縁）との差別化: 今作は**極太ゴシック・機械文書・ネオン**

## 2. カラーパレット（トークン）

| トークン | 値 | 用途 |
|---|---|---|
| `--bg` | `#0A0E1A` | 地。青バイアスの暗色（純黒は使わない） |
| `--cyan` | `#4DEAFF` | 主役。HUD文字・罫線・チップ枠・強調 |
| `--yellow` | `#FCEE0A` | 強調ブロック。**面で使い、上に黒文字**（仮題タグ・ハザードストリップ・コーナーアクセント） |
| `--red` | `#FF2A44` / `#8C0F1E` | **大気専用**。周縁のもや（radial-gradient・低不透明度）とタイトルの色収差。UI部品・文字色には使わない |
| `--text` | `#E9F1F7` | 本文。青みの近白（※初版の`#C9D8E4`は暗背景で暗すぎたため引き上げ。2026-07-13） |
| `--text-strong` | `#F2F7FB` | 見出し・強調 |
| `--text-dim` | `#C6D5E0` / 42%白 | 説明文・英字添え（本文より一段だけ落とす。落としすぎない） |
| `--term` | `#A9E6F8` | **固有用語の弱強調**（canon用語: 旧NET・ミリテク・ノーマッド等）。太字なし・淡シアン。データ記法は `*用語*`。強強調 `**太字**`（白）とは別物 |

**役割分担の原則**: シアン＝線と文字／イエロー＝面（黒文字を載せる）／赤＝大気のみ。この分担を崩さない（赤をUIに使うと警告色と混ざり、イエローを文字にすると視認性が死ぬ）。

## 3. タイポグラフィ

| 役割 | 書体 | 処理 |
|---|---|---|
| タイトル | 游ゴシック 900（極太） | 色収差グリッチ（text-shadow: 赤+シアンを左右に0.14cqw） |
| 本文 | 游ゴシック / メイリオ | 行間 1.9〜2.0・字送りゆったり |
| 英字ラベル | Consolas 等幅 | 大文字・字間 .2〜.4em・シアン or 黄。「MURDER MYSTERY // CYBERPUNK RED」等の飾り英字 |
| 対句タグ | 等幅・シアン・字間広 | 勢力の「頭脳と回路」等 |

- Webフォント非依存（システムフォントのみ）。印刷・画像化・Artifactプレビューのどれでも同じに出る
- 英字を和文の「添え」として使う（fac名の後ろに小さくZETATECH等）。世界観の記号として機能

## 4. モチーフ語彙

- **角欠きブロック**（clip-path で1〜2隅を45°カット）: 2077 UIの署名的形状。タグ・チップに使う
- **HUDフレーム**: 細いシアン枠＋対角2隅だけ黄色い太コーナー
- **スキャンライン**: repeating-linear-gradient の細横縞・低不透明度・全面
- **赤いもや**: 画面外周からの radial-gradient（右上・左下）＋上下の薄い帯。中央の可読域には侵入させない
- **ハザードストリップ**: 黄地×黒斜め縞の帯。注意・宣言系の文言（卓内正史宣言等）の定位置
- **▶ マーカー**: 締め文・重要行の頭

## 5. レイアウト文法（ペライチ系）

- 判型: **横 1600×1000（16:10）**。前作踏襲（オンラインセッションの画面共有・部屋背景に最適）
- 実装: cqw 単位で全比率を組む（コンテナ幅に対して完全スケール）
- 基本構成: 左カラム（タイトル＋本文）57% ／ 右カラム（リスト・データ）43%、下端にハザードストリップ
- 情報の重み: 左＝読ませる／右＝見せる（一覧）／下帯＝ルール宣言

## 6. 適用ルール

- **配布物に載せる本文はシナリオ文書からの逐語抜粋のみ**。言い換え・要約の書き下ろしは禁止、文単位のカットのみ可（2026-07-13 導入シートで要約書き下ろしによる本文乖離が発生→是正。二重管理事故の防止）
- 配布物は全てこの言語で統一（導入シート・ルール説明・HO表紙・証拠カード）。証拠カードは同パレットで「密度高め・もや無し」等の派生を作る
- 単一テーマ（暗色）で確定。ライト版は作らない（印刷時もダーク面で刷る想定）
- 固有名詞表記は 2077日本語版準拠（research/03 §7.5）に従う
- 仮置き名称（タイトル・地区名・警備会社名）が本決定したら、モックの該当文字列を差し替えて再出力
