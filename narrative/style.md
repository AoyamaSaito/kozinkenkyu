# 文体仕様 — 執筆テンプレート

> 本作の文体・形式面の統一仕様。
> 参考資料として提示されたサンプルテキストから形式要素を抽出。
> 内容ではなく**書き方の骨格**を定義する。

## 1. 視点・人称

- **一人称**: 主人公（久世朔）の主観で全てが進行する
- 一人称代名詞は未定（「俺」「僕」等、別途決定）
- **観測者の内面は直接描かれない**。朔が推測・観察する形でのみ表出
- 地の文は全て朔のフィルターを通す

## 2. 行構成・テンポ

### 2.1. 細切れの短文
一文を一段落として独立させ、テンポと緊張を作る：

```
カウンセリング室のドアに鍵がかかっている。

内側から。

俺が閉めたわけじゃない。
```

### 2.2. 体言止め
名詞・名詞句で文を切る。衝撃・静寂・強調に使う：

```
埃。

埃、埃、埃。
```

```
沈黙。

五秒。十秒。三十秒。
```

### 2.3. 短文と長文の交互
スタッカートな短文の連打と、落ち着いた長文の説明を交互に配置してリズムを作る。短文で緊張を張り、長文で弛緩させる。

## 3. 記号の用法

| 記号 | 用法 | 例 |
|------|------|-----|
| 「」 | セリフ（音声として発された言葉） | 「先生、もうどこにも行かないでください」 |
| （） | 主人公の内心・独白 | （落ち着け。お前はプロだ） |
| …… | 短い間、逡巡、言い淀み | 「……入っていいよ」 |
| ………… | 長い沈黙、重い空白 | 「…………」 |
| —— | 思考の飛躍、挿入、ドラマチックな転換 | その推しが。——目の前に。——いる。 |
| --- | シーン区切り（場面転換） | 水平線として使用 |
| ◇ | 大きな場面転換（時間跳躍等） | 回想への移行等 |

### 3.1. ——（ダッシュ）の詳細
最も多用される記号。以下の機能を持つ：
- **割り込み**: 思考の途中に別の認識が入る
- **補足挿入**: 本筋の中に補足情報を挟む
- **転換**: 場面・意識の急な切り替え
- **余韻の代替**: 「……」が間なら「——」は跳躍

## 4. モノローグの二重構造

### 4.1. 地の文（主人公の語り）
括弧なし。主人公の主観と観察が混在する：

```
前髪の隙間から覗く瞳は、暗い湖のような色をしていた。
```

### 4.2. 括弧付き内心（）
主人公が「自分に話しかける」ときに使用。自嘲、ツッコミ、専門的分析など：

```
（これは……場面緘黙、か……？）
```

```
（落ち着け朝霧蓮。お前はプロだ。深呼吸だ。吸って、吐いて——）
```

### 4.3. 使い分け
- 地の文: 状況認識、描写、冷静な思考
- （）: 動揺、自嘲、感情的反応、専門知識の独白

## 5. セリフの配置

### 5.1. セリフ→地の文→セリフ の呼吸
セリフの後に、主人公のフィルターを通した解釈・描写を挟む。連続セリフは避ける：

```
「先生はあたしのだ。誰にも渡さない」

次に聞こえたのは、いつも強がって笑っていた少女の声。
その声はもう笑っていなかった。
```

### 5.2. セリフなしの「声」の描写
言葉を直接引用せず、声の質感や印象だけを描くことも可：

```
三番目の声は、かすかすぎて空気に溶けそうだった。
```

## 6. 時間・場面処理

### 6.1. 時間経過の省略
地の文で端的に宣言する：

```
三時間後。

カウンセリング室は、まあ、なんとか人が使える状態になっていた。
```

### 6.2. 回想への入り方
——（ダッシュ）で導入する：

```
——あの日に戻ろう。全てが始まった、春の日に。
```

### 6.3. 場面転換
`---`（水平線）で区切り、必要に応じて `◇` を挟む。

## 7. 本作への適用ノート

### 7.1. 観測者の声の描写（Phase 0〜2）
- Phase 0-1: 気配のみ。地の文で「何かが居る」の感覚を描写
- Phase 2: 声だけの交流。セリフは「」だが、声の出所が不明な描写を添える
- Phase 3以降: 姿が見え、通常の対面会話形式に遷移

#### 7.1.1. プロローグでの先出し運用
物語冒頭のプロローグでは、**Phase 2相当の声の交流を時系列に先立って提示**する（`arcs.md` §6.1 参照）。このとき：
- 声は「」で引用する（Phase 2ルール準拠）
- 声の出所描写は「空気がそのまま震えたような、方向の定まらない声」等の定型で統一し、プロローグと Phase 0 ラストで**同一表現を反復**する。反復自体が遡及接続の装置になる
- 視覚情報（髪・目・服装・身長・年齢）は**絶対に描写しない**。Phase 3 の開示まで温存
- 朔の内心で「女性の声」程度までは識別してよい。それ以上の属性は Phase 3 まで伏せる

### 7.2. 観測者のセリフのトーン
観測者の発話は `vocabulary.md` に準拠。主人公のフィルターを通すため、冷たさ・毒舌の印象が朔の地の文で増幅される。

### 7.3. ホラー描写（Phase 0）
短文・体言止め・沈黙の数値描写（「五秒。十秒。」）を多用。長尺で引っ張る構成。

### 7.4. 避けるべきこと
- 太字・斜体等のフォント装飾（プレーンテキストで完結させる）
- 観測者の内面の直接描写（一人称制限を厳守）
- 地の文での説明過多（朔は聡明だが、読者に向けた解説者にならない）
