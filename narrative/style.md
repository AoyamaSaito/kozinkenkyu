# 文体仕様 — 執筆テンプレート

> 本作の文体・形式面の統一仕様。
> 参考資料として提示されたサンプルテキストから形式要素を抽出。
> 内容ではなく**書き方の骨格**を定義する。

## 0. 文体の肌感と参照

### 0.1. 媒体想定
- **想定掲載先**: カクヨム／小説家になろう（Web小説）
- **読者想定**: スマートフォンでスクロールしながら読む、日常の合間の可処分時間
- **文体のラベル**: **静謐系ライト** — ライトノベルとしての読みやすさを持ちつつ、一人称の思考密度は保つ

### 0.2. 参照作風
- **近い作風**: 三秋縋（「スターティング・オーヴァー」「恋する寄生虫」）、河野裕（「いなくなれ、群青」シリーズ）、杉井光、入間人間の静謐系
- **距離を取る方向**: 純文学（カズオ・イシグロ、小川洋子）、ハードボイルド、なろう系パワーインフレ
- **核の感覚**: 「一人称の内面が持つ熱量を保ったまま、読みやすく、スマホで指が止まる窓を作る」

### 0.3. 「純文学ノイズ」として避けるもの
以下は現行で過剰になりがちな要素。**要所では使うが、手癖で量産しない**：
- **倒置法の多用**（例:「この空間も、この人も、余計な色を知らないみたいにできている」）
- **漢語寄りの抽象比喩**（例:「質感の記憶に引っかからない壁」→「触っても冷たくもあたたかくもない壁」）
- **体言止めの詩的連打**（§2.2 の技法自体は残すが、**異常空間の切迫感の演出**のためだけに使い、日常・対話・内省では多用しない）
- **——（ダッシュ）の装飾的多用**（§3.1 の機能用途に限定）
- **一文の冗長化**（形容句を挟みすぎない、主述を遠ざけない）
- **漢語率の偏重**（「均質」「吸収」「反響」等の硬質な語を、和語に置換可能な箇所で和語化する）

### 0.4. シーン別の文体密度
同じ文体仕様内でも、シーンのテンションで密度を変える（§2.5 の拡張）：

| シーン | 文体の方向 | ノイズ要素の許容度 |
|--------|-----------|------------------|
| 日常・現実A/B/C... | 口語寄り、柔らかく | 低（手癖を封印） |
| 観測者との対話 | やや硬質、リズム重視 | 中 |
| Visit 1・異常空間 | 体言止め・反復の許容度最大 | 高（意図的な過剰は OK） |
| 帰還直後・気付き | ゆっくりした内省、空行多め | 低〜中 |

## 1. 視点・人称

- **一人称**: 主人公（久世朔）の主観で全てが進行する
- 一人称代名詞は未定（「俺」「僕」等、別途決定）
- **観測者の内面は直接描かれない**。朔が推測・観察する形でのみ表出
- 地の文は全て朔のフィルターを通す

## 2. 行構成・テンポ

### 2.1. 細切れの短文
一文を一段落として独立させ、テンポと緊張を作る：

```
カウンセリング室のドアに鍵がかかっている。

内側から。

俺が閉めたわけじゃない。
```

### 2.2. 体言止め
名詞・名詞句で文を切る。衝撃・静寂・強調に使う。**異常空間の演出（Visit 1 の廊下描写等）で特に効果を発揮する技法。全編で多用しない**：

```
沈黙。
```

```
白い壁。白い天井。白い床。
廊下。長くまっすぐな、廊下。
```

```
分岐。
——さっきは、なかった。
```

#### 2.2.1. 単語反復による体言止め
同一語を繰り返して空間・視界の均質さや反復感を表す：

```
左を見る。白。右を見る。白。振り返る。同じ廊下が、奥まで続いている。
```

```
右を選ぶ。どちらを選んでも同じだろうという諦めが、手を先に動かした。白。左。白。
```

#### 2.2.2. 体言止め＋一語文の連打
数値や状態の変化を一語で並べる：

```
沈黙。

五秒。十秒。三十秒。
```

### 2.3. 断定→否定の反復パターン
動作や状態を一度断定し、次行で「——はずだった」「——はずが、〜」で反転させる。現実の輪郭が崩れる感覚を生む、本作固有の駆動装置：

```
足を止めた。
——止めた、はずだった。
```

```
僕は学校からの帰り道にいたはずだ。
いた、はずだった。
```

```
廊下は真っ直ぐだった——はずが、角がある。
```

### 2.4. 短文と長文の交互
スタッカートな短文・体言止めの連打と、落ち着いた長文の描写を交互に配置してリズムを作る。短文で緊張を張り、長文で弛緩させる。観測者の部屋（定常シーン）では長文比率を上げ、Phase 0 の廊下描写では短文比率を上げる。

### 2.5. 地の文のブロック化（可変密度）
**シーンのテンションに応じてブロック密度を切り替える**。均一リズムにしない。

| シーン種別 | 推奨形式 | 例 |
|-----------|---------|-----|
| 定常・観測者との対話 | 3〜5行の連続ブロック＋空行 | Ep1 冒頭〜セクション 0 |
| ホラー・孤立・異常空間 | 1〜2行ブロックを空行で断片化 | Ep1 セクション 1〜4 |
| 日常への帰還 | 3行前後の中庸ブロック | Ep1 セクション 5 |

共通ルール：
- **セリフ「」『』・内心（）の前後**には空行を入れる（呼吸）
- **同じ思考・動作・描写の流れ**は改行なしで詰める
- ブロック間には空行を入れて、リズムを分節する
- ホラーパートでの一文独立は推奨。定常パートでは多用しない

## 3. 記号の用法

| 記号 | 用法 | 例 |
|------|------|-----|
| 「」 | セリフ（音声として発された言葉） | 「先生、もうどこにも行かないでください」 |
| 『』 | 姿の見えない観測者の声（Phase 1〜2） | 『質問しているのは私だ。立場を弁えろ』 |
| （） | 主人公の内心・独白 | （落ち着け。お前はプロだ） |
| …… | 短い間、逡巡、言い淀み | 「……入っていいよ」 |
| ………… | 長い沈黙、重い空白 | 「…………」 |
| —— | 思考の飛躍、挿入、ドラマチックな転換 | その推しが。——目の前に。——いる。 |
| --- | シーン区切り（場面転換） | 水平線として使用 |
| ◇ | 大きな場面転換（時間跳躍等） | 回想への移行等 |

### 3.1. ——（ダッシュ）の詳細
最も多用される記号。以下の機能を持つ：
- **割り込み**: 思考の途中に別の認識が入る
- **補足挿入**: 本筋の中に補足情報を挟む
- **転換**: 場面・意識の急な切り替え
- **余韻の代替**: 「……」が間なら「——」は跳躍

## 4. モノローグの二重構造

### 4.1. 地の文（主人公の語り）
括弧なし。主人公の主観と観察が混在する：

```
前髪の隙間から覗く瞳は、暗い湖のような色をしていた。
```

### 4.2. 括弧付き内心（）
主人公が「自分に話しかける」ときに使用。自嘲、ツッコミ、専門的分析など：

```
（これは……場面緘黙、か……？）
```

```
（ここ、どこだ）
```

```
（だいたい、そんなもんだ）
```

### 4.3. 使い分け
- 地の文: 状況認識、描写、冷静な思考、距離を取った自己分析
- （）: 動揺、自嘲、短い実感、専門知識の独白
- 内心（）は**短く切る**のが基本。長い独白にしない（朔の語りは乾いた知性寄りで、感情を垂れ流さない）

## 5. セリフの配置

### 5.1. セリフ→地の文→セリフ の呼吸
セリフの後に、主人公のフィルターを通した解釈・描写を挟む。連続セリフは避ける：

```
「先生はあたしのだ。誰にも渡さない」

次に聞こえたのは、いつも強がって笑っていた少女の声。
その声はもう笑っていなかった。
```

### 5.2. セリフなしの「声」の描写
言葉を直接引用せず、声の質感や印象だけを描くことも可：

```
三番目の声は、かすかすぎて空気に溶けそうだった。
```

## 6. 時間・場面処理

### 6.1. 時間経過の省略
地の文で端的に宣言する：

```
三時間後。

カウンセリング室は、まあ、なんとか人が使える状態になっていた。
```

### 6.2. 回想への入り方
——（ダッシュ）で導入する：

```
——あの日に戻ろう。全てが始まった、春の日に。
```

### 6.3. 場面転換
`---`（水平線）で区切り、必要に応じて `◇` を挟む。
- `---`: 同一エピソード内の小場面転換（数秒〜数分の経過、視点の切替、廊下の角を曲がる等の空間断絶）
- `◇`: 大きな時間跳躍（プロローグ→回想本編、等）
- Ep1 では冒頭の現在パート→`◇`→Phase 0 過去回想、という構造で用いている

### 6.4. エピソード内セクション番号
1エピソードは複数のセクションで構成し、`## 1` `## 2` … と見出し番号のみで区切る。
- サブタイトルは付けない（朔の語りに章題的な客観性を持ち込まない）
- セクション 0 に相当する導入部は見出しを付けず冒頭に直接置く（Ep1 の冒頭部 参照）
- セクション内でさらに `---` による小場面転換が入る

## 7. 本作への適用ノート

### 7.1. 観測者のセリフ表記
- **Phase 0**: 気配のみ。声は聞き取れない。地の文で「何かが居る」の感覚を描写
- **Phase 1〜2**: 声だけの交流。セリフは**『二重鉤括弧』**で表記し、声の出所が不明な描写を添える
- **Phase 3以降**: 姿が見え、通常の**「」**で表記。対面会話形式に遷移

#### 7.1.1. 『』と「」の使い分け

| Phase | 観測者の状態 | 表記 | 例 |
|-------|-------------|------|-----|
| 0 | 気配のみ（声は不明瞭） | 地の文のみ | 誰かが——僕の背中に向かって、何かを言ったのだ。 |
| 1〜2 | 声のみ（姿は見えない） | 『』 | 『質問しているのは私だ。立場を弁えろ』 |
| 3以降 | 姿が見える | 「」 | 「質問しているのは私だ。立場を弁えろ」 |

- 朔のセリフは全Phase共通で「」を使用
- 『』は**観測者の声が空間に直接響く異質さ**を視覚的に表す装置。姿の開示（Visit 6）で「」に切り替わる
- Visit 6内では、姿が見えた瞬間を境に『』→「」へ遷移する

#### 7.1.2. 声の質感の定型描写
声の出所描写は「空気がそのまま震えたような、方向の定まらない声」を定型とする。Phase 0ラストで初出し、Phase 1〜2の声のみ期間で反復する。この反復が遡及接続の装置になる。

#### 7.1.3. プロローグの運用
物語冒頭のプロローグは**Phase 4相当の対面シーン**を時系列に先立って提示する（`arcs.md` §6.1 参照）。このとき：
- 観測者は視覚的に描写する（白髪・白衣・姿勢・身振り）
- セリフは「」で表記（Phase 3以降のルール準拠）
- 研究室的空間に二人がいる定常状態を描き、読者に「この関係性はどう始まったのか」という問いを持たせる
- プロローグ→◇→Phase 0（第1回来訪）で時系列が巻き戻る構造

### 7.2. 観測者のセリフのトーン
観測者の発話は `vocabulary.md` に準拠。主人公のフィルターを通すため、冷たさ・毒舌の印象が朔の地の文で増幅される。

### 7.3. ホラー描写（Phase 0）
以下の技法を組み合わせて、均質な白い廊下の異常さを段階的に浸透させる：
- **体言止めの連打**（§2.2）：「白い壁。白い天井。白い床。」
- **単語反復**（§2.2.1）：「白。白。」「白。左。白。」
- **断定→否定の反復**（§2.3）：「止めた、はずだった」
- **感覚の欠落の列挙**：革靴の音が反響しない／風が頬に当たらない／息が切れない／時計の秒針が動かない、等を一文ずつ独立させる
- **1〜2行ブロック＋空行**で断片化（§2.5）
- **最後に「背後で何かがした」で声の初出を仕込む**（§7.1.2 の定型描写を用いる）

### 7.4. 避けるべきこと
- 太字・斜体等のフォント装飾（プレーンテキストで完結させる）
- 観測者の内面の直接描写（一人称制限を厳守）
- 地の文での説明過多（朔は聡明だが、読者に向けた解説者にならない）
- **漢語偏重・抽象比喩への逃避**: 「質感の記憶」「意識の輪郭」のような詩的抽象は、意味より雰囲気が立ってしまう。和語で具体化する
- **倒置・体言止めの手癖化**: 感覚を刻む効果は強いが、**全段落で使うと媒体のテンポを壊す**。§0.4 のシーン別密度に従う
- **比喩の装飾化**: 比喩は「より分かりやすくするため」に使い、「雰囲気を増やすため」に使わない

## 8. 確定項・未確定項

### 確定済み
- 朔の一人称代名詞 → **「僕」**
- 朔の語り口調 → **乾いた知性寄り**。感情を直接吐露せず、断定と否定の反復、体言止めで内面を間接的に漏らす。内心（）は短文で切る
- 声の質感の定型描写 → **「空気がそのまま震えたような、方向の定まらない声」**（Phase 0 ラスト初出／Phase 1〜2 で反復）

### 未確定
- Visit 6（姿の開示）における『』→「」遷移の具体的な一文設計
- 観測者視点の部分的混入可否（現時点では禁止。Phase 後半で解禁検討の余地あり）
