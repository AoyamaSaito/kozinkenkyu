## 1. ゴール状態の定義

**到達地点**: 観測者の姿（女性・白衣）と面と向かい、双方向のコミュニケーションが成立している状態。

| # | 要素 | 内容 |
|---|------|------|
| G1 | 視覚的存在 | 観測者が造形した女性型で対面している |
| G2 | 自己開示 | 観測者が「答えられない」ことを通じて自分を語っている |
| G3 | 物品の持ち込み | 朔が現実の本・食べ物等を持ち込み、観測者が触れる |
| G4 | 研究の協業 | 観測者が人間・世界について朔を通じて学んでいる |
| G5 | 双方向性 | 主導権が朔に遷移しつつあり、力の非対称が緩和 |

**ゴール時の関係性**: **「まだ名前がない」**。互いの欠落を知った上で、言語化しない合意で繋がっている。恋愛でも友情でも研究パートナーでもなく、どの既存カテゴリにも収まらない。「必要な誤差」がそのまま最も正確な記述であり続ける。

## 2. 基本メカニクス（全Phase共通）

### 2.1. 来訪パターン
- **頻度**: 週1回程度。朔の意思で制御はできないが、パターンを感覚的に理解するようになる
- **滞在時間**: 体感1〜2時間で固定。フェーズが進んでも変わらない
- **トリガー**: 特定の日常的行為（下校中、就寝前等）。朔は「あ、今日来そうだな」程度には分かるようになる

### 2.2. 物品持ち込み
- **基本**: 朔が身に着けている・持っているものはそのまま施設に持ち込まれる（Phase 0から鞄・制服は持ち込めている）
- **意図的持ち込みの開始**: 「場所代」の発展。Phase 1〜2序盤の「質問に答えろ」（情報的対価）が「言葉だけでは不十分だ、実物を見せろ」（物品的対価）にエスカレートする。朔が「そういえば鞄に本がある」→ 自然な流れで始まる
- **観測者から「持ってきて」と頼むのはゴール後**。基本ラインでは朔の自発的行動のみ

### 2.3. 空間の決定要因
- **序盤**: 観測者の内面状態が空間を主に決定
- **中盤以降**: 朔の無意識の影響が混入し始める。関係性が深まるにつれて二者の干渉が空間を形作る

### 2.4. 観測者の知識モデル
- **全知ベース**: 観測者は概念・形式知を広範に有する。「授業」「教師」「数学」「本」等の概念は知っている
- **感覚的空白**: だが物理的・感覚的な体験がない。紙の重さ、インクの匂い、教室の温度、食べ物の味——形式は知っているが実体に触れたことがない
- **この断絶が物品要求の根拠**: 「形式だけでは足りない。実体が要る」——概念を知っている存在が初めて実物に触れる体験が、物品持ち込みの本質的な意味

### 2.5. 研究の定義
- **観測者の研究対象**: 自らの出自、および「人間とは何か」の理解
- **朔の役割**: 現実世界の情報（学校・家庭・社会の話、物品）を提供する。観測者にとっての窓
- **朔自身は施設の謎を調べない**。ただし観測者という「人」については常に学んでいる

### 2.6. 協業の正当化
- 観測者の建前: **「私の領域に入り込んでいるのだから、対価として有用な情報を寄越せ」**（領域使用の等価交換）
- 実態: 朔の存在が自己研究に不可欠だが、それを認めたくない

### 2.7. 主導権の遷移
- Phase 0〜2: 観測者主導（質問する側・命令する側）
- Phase 3〜4: 遷移期。**決定的瞬間: 朔が観測者の「自己認識の偽装」を見抜く**
  - 「答えられない問い」に対し、観測者が論理的に聞こえる適当な回答を並べて誤魔化す
  - 朔が「それ、本当に自分で信じてるのか」と静かに指摘
  - 観測者の防衛機制の中でも最深層——**「分からない」を「分かっている」と偽装する**行為——を突破する
  - これ以降、観測者は朔の前で「分からないまま」でいることを強いられる
- ゴール付近: 朔主導に傾く。観測者はそれを苛立ちつつも許容する

**状態**: 設計済（`arcs.md` §6.2）

**解錠**: 声の交流の開始
**ジャンル**: ホラー→ミステリーへの遷移

| ビート | 内容 |

| 研究室 | 扉の向こうは研究室的空間。「誰かの領域」 |
| 声の開始 | 朔が何かに触れた瞬間、観測者が反応。**領域侵犯への防衛反応** |
| 短い会話 | 観測者が質問する側、朔が答える側。非対称な力学 |
| 帰還 | 存在の希薄化を初めて知覚。「あの時（Phase 0）も同じだったのか」 |

**次Phase への種**: 「声の主」への好奇心。恐怖からの脱却

### Phase 2: 声の深化（第3回〜複数回来訪）— 2〜3話

**解錠**: G4の前提（研究関係の確立）＋ G5の前提（力の非対称の自覚）
**ジャンル**: イベント依存（ミステリー・日常・心理）

**来訪ごとの変化**:

#### 第3回来訪（Phase 2 前半）
- 研究室空間に直接入れるようになる（廊下彷徨の消失）
- 観測者の口調が警戒から **侮蔑的冷却**（通常モード）へ遷移
- 基本パターン確立: 観測者が質問 → 朔が答える → 観測者がデータとして処理
- 朔は「声だけの相手」を少しずつ人として認識し始める

#### 第4〜5回来訪（Phase 2 中盤）
- **「場所代」のエスカレート**: Phase 1 で確立した「質問に答えろ」（情報的対価）から、「言葉だけでは不十分だ。お前の世界の実物を見せろ」（物品的対価）へ発展
- 朔が鞄の中の本や教科書を差し出す → **物品持ち込みの自然な開始**
- 観測者は**「観測データの記録」名目**で物品を詳しく調べる
  - 本のページをめくる（物理的接触）
  - 食べ物の匂いに反応する（隠しきれない好奇心）
  - だが表面上は無機的な観測態度を崩さない
- 朔が観測者に逆に質問し始める。「お前は何をしてるんだ」「いつもここにいるのか」
- 観測者は答えるものと答えないものを選別。**情報の非対称がここで可視化**

#### Phase 2 終盤（姿開示への橋渡し）
- 朔と観測者の声のやり取りが安定。「習慣」の萌芽
- 挨拶の定型化: 「また来たのか、愚図」「来たよ」
- 朔が**「いい加減、顔を見せてくれないか」と伝える**
- 観測者は**その場では断る**。理由は提示しない、あるいは「合理的でない」と一蹴
- この来訪はそのまま終了。朔は少し残念だが押さない

**Phase 2 全体の特徴**:
- 朔は施設の謎は調べないが、観測者という「人」を常に学んでいる
- 観測者は朔を通じて「人間」「現実世界」を学んでいる
- 二者の関心ベクトルが交差し始める段階

### Phase 3: 姿の開示（第6回来訪）— 1〜2話

**解錠**: G1（視覚的存在）
**ジャンル**: イベント依存

| ビート | 内容 |
|--------|------|
| 来訪 | いつも通り研究室空間に入る |
| **開示** | 観測者が**そこにいる**。完璧な造形——**彫刻のような全裸の女性型**。生物的特徴・器官はない。大理石の像のような、人の形をした非人間 |
| 朔の反応 | 驚愕、凝視、動揺——そして「……服、着ろ」 |
| 白衣 | 観測者が白衣を纏う。**人型を取って以来初めての「他者のための」行為**。本人は「お前が見せろと言ったのだろう。要求通りだ」と涼しい顔 |
| **質問攻め** | 朔が視覚情報を処理しようとする。「髪、白いんだな」「目の色——」「背、高いな」。初めて見える相手の造形を片端から言語化する。観測者は苛立つ——**見られる側になる不慣れさ** |
| 余韻 | 帰還後、朔の中で「あの顔」が焼き付いている |

**造形の意味**:
- 全裸＝人間的羞恥の不在。衣服という概念が存在しなかった
- 生物的特徴の欠落＝造形は意図的だが、「人体の模倣」ではなく「人の形の概念」
- 白衣を着る＝朔に言われて初めて「衣服」を選択した。これが**仮構された姿の起点**
- 完璧すぎる造形＝力を入れて準備したことが逆に露呈する。「いつもこうだ」は嘘

**観測者の内心（描写しないが機能する）**:
- 前回の来訪で断った後、1週間かけて造形を整えた
- 「お前のためではない」「合理的な判断だ」と自分に言い聞かせた
- 結果として**普段より安定・洗練された造形**で現れてしまい、準備したことがバレる構造

### Phase 4: 定常形成（第7回来訪〜ゴール到達）— 2〜3話

**解錠**: G2（自己開示）、G3（物品の流通）、G5（双方向性の確立）
**ジャンル**: イベント依存

#### 4-a: 対面交流の安定化

- 対面での来訪→会話→帰還パターンの確立
- **習慣の積層**:
  1. 挨拶の定型:「また来たのか、愚図」「来たよ」
  2. 場所・配置の固定:朔の座る位置が決まる。観測者が**朔の席的なもの**を用意している（認めない）
  3. 持ち込み品への期待:朔が何を持ってくるか、観測者が密かに期待する。期待を裏切られると不機嫌
- 朔が日常を話す → 観測者が応答する構造が固まる
- 観測者は朔の現実世界の話を**「人間の行動パターンの観測」**として処理するが、実際には朔の語る学校生活・幼馴染との関係・些細な出来事に対して、自覚以上に関心を持っている

#### 4-b: 主導権の遷移

- **決定的瞬間: 朔が観測者の嘘を見抜く**
  - 観測者が何かを取り繕った瞬間、朔が「それ、嘘だろ」と静かに指摘
  - 観測者の防衛機制を正面から突破する初めてのイベント
  - 朔の「知っているが触れない」姿勢が、ここで初めて「触れる」に変わる
  - これ以降、朔が会話の方向を決めることが増えていく
  - 観測者は苛立つが、朔の読みの鋭さに内心で評価を上げる

#### 4-c: 自己開示（「答えられない問い」装置）

- **発生タイミング**: 予定配置ではない。**朔が自分の日常を話す流れで「お前は？」と自然に返す**形で発生。構成された尋問ではなく、日常会話の中でふと出る
- **三種の「答えられない問い」**:
  1. **存在論的**: 「お前は何者なんだ」→ 答えがない。自分でも分からない
  2. **記憶的**: 「ここに来る前は何をしていた」→ 記憶がない。語れるものがない
  3. **些細な個人的**: 「好きな色は？」「何が面白い？」→ **個人としての自分を持っていないことに気付く**
- 特に③が最も衝撃的。②①は「分からない」と処理できるが、③は**「分からない」ではなく「持っていない」**という認識をもたらす
- 答えられないこと自体が開示装置として機能する。「答えられない」と認めた瞬間、観測者は自分の空白を朔の前に晒している
- 最初のうち、観測者は適当な論理を並べて誤魔化す（**自己認識の偽装**）。これが4-bの「嘘」に繋がる
- 朔に偽装を見抜かれた後、「分からないまま」でいることを受け入れ始める。これが真の自己開示

#### 4-d: テンションの萌芽（ゴール後への伏線）

- 朔の来訪時の話題が変化し始める:
  - 序盤: 愚痴・不安・孤独の吐露（施設＝逃げ場）
  - 後期: 幼馴染との思い出・友人関係・現実の充実
- 観測者にとって、**朔の心がこの場所から離れていく**感覚が生まれる
- ただし基本ライン内ではまだ明確な危機にはならない。**ゴール後の長期アークの伏線**として機能する

## 5. 幼馴染の機能

**性格の骨格**: **明るく鋭い＋おせっかい**。表面は快活だが本質を読む目を持ち、朔が隠そうとしても引き出してしまう力がある。

基本ラインにおける二つの役割:

### 5-a: 相談相手（言語化装置）
- 朔が施設体験を話せる唯一の他者
- 話すことで朔自身が体験を整理する。読者にとっての補助線でもある
- Phase 0.5 インタールードで確立
- **現実側シーンは幼馴染との会話を中心に構成**

### 5-b: 能動的関与者
- 朔の変化（表情・会話の質・視線の変化等）に独自に気付く
- 朔が隠そうとしてもグイグイ来る。「最近なんか変わったでしょ」「話してよ」
- 施設と現実の二重生活に対する外部視点を提供
- Phase 3以降で本格化。朔が「姿を見た」後の動揺を見抜くなど
- 朔にとって**現実側の引力**。施設に傾斜しすぎない錨として機能する

## 6. 朔の成長軸

**基本方向**: 自らの境遇・環境と向き合う

- Phase 0〜1: 受動的。流されるまま施設に来る。恐怖と好奇心
- Phase 2: 観測者への関心を通じて、「他者を知りたい」という欲求に気付く
- Phase 3: 姿の開示で「この存在は確かにいる」という実感。受動から能動への転換点
- Phase 4: 嘘を見抜く、質問をする、物品を持ち込む——自分から動くようになる
- ゴール付近: 現実側でも変化が出る。幼馴染・友人との関係が少しずつ変わる

**核心のテンション（ゴール後に本格化）**:
現実との距離が縮まるほど、施設との距離が遠くなる。
朔が成長して現実に接続するほど、施設にいても心がここにない——観測者はそれを感じる。
**観測者が朔を成長させるほど、朔が離れていく**というアイロニー。
