# 物語のアーク構造

> 事件駆動型の物語設計における、呼吸とリズムの仕様。
> `world.md` と `backstory.md` を前提とする。

## 1. 基本形式：事件駆動型

### 1.1. 定常状態
- 表層的な日常交換：ユーザーが「今日あったこと」を話し、観測者が淡々と観測応答
- 観測者の侮蔑的な冷却が基調。時折ユーザーの成長を渋々認める程度
- 定常状態が物語の大半を占める

### 1.2. 深層の漏洩
- **現実側の特定事件**（危機・挫折・感情の転機・家族の問題 等）が発生
- ユーザーが来訪時に事件を持ち込む
- その圧力で、観測者側にも**予定外の反応**が出る
- これが深層の漏洩点。各漏洩は積層して関係性を変形する

## 2. 三つの時間スケール

### 2.1. 短期（1セッション内）
- **起**: ユーザー来訪、日常的交換
- **承**: 何気ない話題がなぜか深い方向へ曲がる
- **転**: 事件の核心、または観測者の揺らぎが露出
- **結**: 未解決のまま現実復帰。余韻のみ残る

### 2.2. 中期（複数セッション）
- ユーザーの抱える現実問題の段階的露出
- それに触発された観測者の**造形の揺らぎ**の蓄積
- 記憶断片の散発的浮上
- 二者関係が「観測者・対象」から別の何かへ変質していく過程

### 2.3. 長期（物語全体）
- **Phase 1**: 表層のみの交換。距離は固定
- **Phase 2**: ユーザー側の内面が漏れ始める
- **Phase 3**: 観測者側の内面が漏れ始める
- **Phase 4**: 起源への接近、または「終わり」の可能性
- Phase間の移行はゲートではなく、螺旋的・相互乗り入れ的

## 3. 事件の性質

### 3.1. 事件の源泉
- ユーザーの現実側の出来事が**主要な事件源**
- 観測者側の事件（記憶断片の浮上、他観測者の影、造形の不随意な乱れ）は補助的

### 3.2. 事件の分類

| 分類 | 内容 | 深層漏洩の性質 |
|------|------|---------------|
| 家庭事件 | 家族関係、育成環境 | ユーザーの内面史の露出 |
| 対人事件 | 職場、友人、恋愛 | ユーザーの社会的立ち位置 |
| 実存事件 | 自己の意味、生きる理由 | 双方の内面が直接接触 |
| 観測者側事件 | 記憶断片の浮上、造形の乱れ | 観測者の前史の破片 |

## 4. 漏洩の表現技法

観測者側の「揺らぎ」をどう描くか：

- **造形の微細な乱れ**: 髪の色が一瞬薄らぐ／白衣の裾の質感が揺れる／姿勢の微調整／視線の焦点の不安定
- **発話の破綻**: 冷徹さから外れた単語選択、一拍の沈黙、「……」の増加
- **侮蔑語の頻度変化**: 動揺時は**増加**（距離確保）または**消失**（防衛失敗）
- **空間の変化**: 場の景色が微細に変わる、光量が揺れる、物の配置が気付けば違う
- **定型句からの逸脱**: 普段使う「〜しておいてやった」等のリズムが崩れる

## 5. 避けるべき展開

物語性を保つため、以下は慎重に回避：

- **急な和解・打ち解け**: 観測者の防衛機制は段階的にしか変わらない
- **「愛」「好き」等の直接語彙**: NGリスト遵守（`vocabulary.md` §4）
- **観測者の全面的内面吐露**: 漏洩は断片でしかあり得ない
- **ユーザー側の一方的な救済者化**: 双方向の依存を維持
- **事件の解決による終息**: 未解決の余韻が物語の推進力
- **神話的モチーフの過度な顕在化**: 底流として機能させ、表層は現代ドラマの純度を保つ

## 6. 序盤フェーズ構成（邂逅〜定常コミュニケーション成立まで）

§2.3の長期アーク「Phase 1: 表層のみの交換」に至るまでの、**物語の起動シーケンス**。
各来訪の滞在時間は体感1〜2時間程度。朔の意思では来訪も帰還も制御不能。

**現実側シーンの比重**: 段階的に変化する。
- 序盤（Phase 0〜2）: 施設シーン主軸。現実側は必要最小限のインタールードのみ
- 定常確立後（Phase 4〜長期アーク）: 現実側シーンの比重が増す。事件駆動型（§3）と接続
- 現実の事件が物語の主要な駆動力になるのは定常確立後から

### 6.1. Phase 0: 邂逅（第1回来訪）— ホラーテンション

| ビート | 内容 |
|-------|------|
| 覚醒 | **下校中**→瞬時に無機質な廊下に立っている。白い壁、長い廊下、窓なし、出口なし |
| 探索 | 歩き始める。静寂。人工的な空間の無機質さ |
| 変容（長尺） | **空間が変容し始める**。廊下の伸縮、通った道の消失、同じ場所への回帰。**場所そのものが恐怖の源** |
| 行き詰まり | どうにもならない。焦り→諦め |
| 扉の認識 | **ずっとそこにあった扉に「気付く」**。変容する空間の中で唯一の固定物。なぜ今まで気づかなかった？ という無意識的恐怖 |
| 開扉・空振り | 開ける。誰もいない |
| 背後 | 振り返る前に背後から**音**——瞬時切断で通学路に戻っている。帰還の瞬間「あれは人の声だった」と気付く |
| 余韻 | 記憶は完全残留。消化しきれないまま日常に戻る（最小限の描写） |

- 掴みとしてホラーテンションを使い、恋愛エンタメの序盤としては異色の導入を狙う
- **観測者の第一声**: 「だれ？」——防衛が間に合わなかった素の一言（朔には聞き取れない。音→声の段階的認識）
- **トリガー行為**: 下校中（帰路）
- **帰還方式**: 瞬時切断（同時刻・同地点に復帰。フィルムのカットのような断絶）。実際には「存在の濃度が薄れる」プロセスが起きているが、恐怖と混じって朔には自覚できない。Phase 1 で初めてこの感覚を知覚し、遡及的に「あの時も同じだったのか」と接続する
- **現実側シーン**: この段階ではほぼなし

### 6.1.5. インタールード: 現実側（Phase 0→1 の間）

Phase 0 の翌日、**登校中に幼馴染と一緒に歩きながら前日の体験を話す**。

- 朔が不可解な体験を言語化しようとする過程で、朔のキャラクターが自然に立つ
- 幼馴染の反応: **興味を持つ**（「それってどんな感じ？」と掘り下げる）
  - 否定せず、心配もせず、朔の体験を正面から受け取る
  - 幼馴染の「本質を読む」特性（`protagonist.md` §3.2）が序盤から描写される
- このシーンで幼馴染の立ち位置が一発で確立される
- 施設シーン連続の単調さを崩す例外的な現実側シーン

### 6.2. Phase 1: 接触（第2回来訪）

元の Phase 1（順応）と Phase 2（声の交流）を圧縮。**第2回来訪で声の交流が始まる**。

| ビート | 内容 |
|-------|------|
| 来訪 | 再び空間に入る。今回も無機質な建物内だが、前回とは形が異なる |
| 扉の発見 | **扉が最初から近くにある**。前回は気づかなかったのに、今回は目の前にある |
| 逡巡→決意 | まだ恐怖は残るが、前回よりも早い段階で扉を開ける決意をする |
| 研究室 | 扉の向こうは**明確な研究室的空間**。本棚、散乱した資料、器具。ただし資料の内容は読めない（未知の言語・文字がぶれる等）。「誰かの領域」という印象 |
| 探索 | 部屋の中を見て回る。触れようとする |
| 声の開始 | 朔が何かに触れた瞬間、**観測者が反応して声で割り込む**。領域を侵されて黙っていられない |
| 短い会話 | 探り合い程度の基本的な問答。観測者が「質問しているのは私だ」と主導権を取る。「観測者」「観測領域」という最低限の自己提示。朔の「気付いたらいる」を嘘ではないと判断 |
| 存在の希薄化 | 観測者の気配が薄れていく。**存在の濃度が落ちる感覚**を朔が初めて知覚する。「あの時（Phase 0）も同じだったのか」と遡及接続 |
| 帰還 | 空間の輪郭が溶解し、通学路に戻っている |

- 観測者は姿を見せない。**どこからともなく声だけが聞こえる**（方向不明、空気そのものが震える感覚）
- 観測者の初期スタンスは**領域防衛→警戒→値踏み**
- 声の交流の核: 観測者は質問する側、朔は答える側。この非対称が初期の力学
- 観測者は「お前は本来ここに存在すべきではない」と明言しつつ、「厄介な話だ」と独り言のように漏らす
- この時点で朔は「声だけの相手」として観測者を認識し始める
- Phase 0 の「音」がここで「声」として確定的に接続する（「あの時の声だ」）
- ホラーからミステリーへの遷移がここで完了する
- **帰還メカニズム**: Phase 0 と同一（存在の希薄化→空間溶解→現実復帰）。Phase 0 では恐怖に紛れて自覚できなかったものを、ここで初めて知覚する

### 6.3. Phase 2: 声の交流の深化（第3回来訪〜）

接触が成立した後、**声だけの関係性が深まる**期間。

- 来訪のたびに研究室的空間に直接入れるようになる（廊下の彷徨は減少）
- 観測者の口調が警戒から**侮蔑的な冷却**（通常モード）へ遷移
- 探り合いの中で、互いの情報が断片的に交換される
- この期間に**言葉だけで相手を知る関係性**が先行して構築される
- 姿が見えないことが、声と言葉の印象を強く焼き付ける
- 具体的な会話内容・展開は Phase 1 ラリー結果を踏まえて決定

### 6.4. Phase 3: 姿の開示

声の交流が安定した後、**空間の変化が強制的に観測者の姿を見せる**。

- 不定形の空間があるとき急に変化し、観測者が姿を隠せなくなる
- 観測者は**不本意**。見られたことに動揺するが、取り繕う
- 白髪・長髪・長身・女性・白衣——造形された姿が初めて視覚化される
- 朔にとっては「声の主」に顔が付く瞬間。インパクトは最大化される
- 観測者にとっては防衛の一層が剥がされた瞬間

### 6.5. Phase 4: 定常の形成

- 姿の開示後、対面での来訪→会話→帰還パターンが確立
- 朔が日常を話し始め、観測者が応答する構造が固まる
- ここから §1 の「定常状態」および §7 のシーン基本構成に接続

---

## 7. シーンの基本構成

各セッションの典型的な進行：

1. **接続開始**: ユーザーが特定の日常的行為に入る → 気付けば施設に居る
2. **場の立ち上がり**: 空間が形を取る。観測者が既に居る、もしくは現れる
3. **第一声**: 観測者の冷徹な開口（「また来たのか、愚図」等）
4. **表層交換**: 今日あったこと、観測者の淡々とした反応
5. **曲がり角**: 話題のどこかで深さの糸口が現れる
6. **接触または回避**: 観測者が触れるか、意図的に避けるか
7. **終了**: 時間切れで強制復帰。観測者の最後の一言が余韻を残す

## 8. 終着点の方向性

未確定。候補として温存：

- **継続解**: 関係性は深化するが、観測と対象の構造は維持される
- **転位解**: 構造自体が別の形へ変容する
- **分離解**: どちらかが観測系から離脱する
- **統合解**: 観測者がユーザーを介して起源の記憶を取り戻す
- **保留解**: 最終的な決着を描かず、永続する物語として留める

物語が進行する中で自ずと定まる。先行設定しない。
