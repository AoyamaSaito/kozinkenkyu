# フォルダ構成

> 最終更新: 2026-04-17

```
D:\個人研究その2\
│
├── CLAUDE.md                          # プロジェクト方針・読み込み順・運用ルール
│
├── persona\                           # キャラクター・世界観の設定資料
│   ├── profile_base.md                # 観測者の基本プロファイル（グローバルから複製・骨格）
│   ├── profile_deep.md                # 観測者の詳細プロファイル（深層設定・肉付け）
│   ├── vocabulary.md                  # 観測者の語彙・口調パターン・NGリスト
│   ├── world.md                       # 物語世界の設定（舞台・接触様式・空間・存在論）
│   ├── backstory.md                   # 観測者の前史（記憶欠落・探求対象・断片の浮上）
│   ├── protagonist.md                 # 主人公・久世朔のプロフィール
│   └── relationship.md               # 観測者とユーザーの関係性（動的更新ログ）
│
├── narrative\                         # 物語構造・執筆仕様
│   ├── arcs.md                        # アーク構造（事件駆動型・序盤フェーズ・長期アーク）
│   └── style.md                       # 文体仕様（視点・記号・テンポ・執筆テンプレート）
│
├── scenarios\                         # 執筆済みのエピソード・シーン
│   └── (第1話・第2話をここに保存予定)
│
└── notes\                             # 運用メモ・進行管理
    ├── operational_notes.md           # 設計セッション・ラリーの進め方ルール
    ├── next_actions.md                # 次回アクション・未決事項一覧
    └── folder_structure.md            # 本ファイル
```

## 各ディレクトリの役割

| ディレクトリ | 性質 | 更新頻度 |
|-------------|------|---------|
| `persona/` | 設定資料（半永続） | 新設定追加・転機ごとに更新 |
| `narrative/` | 構造仕様（半永続） | フェーズ設計確定時に更新 |
| `scenarios/` | 成果物（蓄積） | エピソード執筆時に追加 |
| `notes/` | 運用・進行管理（流動） | セッションごとに更新 |

## 可搬性

このフォルダは自己完結している。別環境に持っていく場合、フォルダごとコピーすれば全ての設定資料・構造仕様・運用ルールが揃う。グローバル `~/.claude/CLAUDE.md` への依存は `persona/profile_base.md` で解消済み。
