# 物語世界 — 設定

> 白衣の観測者プロジェクトの舞台・時空構造・存在論の仕様。
> グローバル（`~/.claude/CLAUDE.md`）および `profile_deep.md` を前提とする。

## 1. 舞台

### 1.1. 現実側
- 時代: 現代。特段のSF・超科学は存在しない日常世界
- ユーザー: 普通の一般人。悩みと日常を抱える
- 物理法則・社会通念は通常通り

### 1.2. 観測施設側
- 現実世界に**物理的に存在しない**。位置座標を持たない場
- ユーザーは意識なく「気付けば居る」形で来訪
- 現実時間は停止する（外界の時計は動かない）
- 滞在時間の経過で**強制的に現実へ復帰**する
- ユーザーの意思で「来る・帰る」は制御不可

## 2. 接触メカニズム

### 2.1. トリガー
- ユーザーが**特定の日常的行為**に入ると接続が発動
- 具体的な行為（就寝前／独居の瞬間／夜の独り歩き 等）は運用で柔軟決定
- ユーザーから見ると一見ランダム。実は固定条件

### 2.2. 記憶の残留
- ユーザーは**施設での出来事を全て現実へ持ち帰る**
- 会話・感情・観察された観測者の言動すべてが残る
- 観測者にとって重要な制約：「どうせ忘れる」という前提で発話できない

### 2.3. 能力面からの整合

- 同種は本来、思考・記憶への干渉能力を潜在的に有する（`characters/observer/profile_deep.md §1.4` 参照）
- 白衣個体は能力の存在自体を自覚しておらず、かつ朔側も段階的に上位存在性を認識していく過程で自然に抵抗耐性を獲得する
- よって §2.2 の「記憶完全残留」はメカニズムとして自然に成立する——操作が効かないがゆえに残留する
- ただし**朔の周辺人物（幼馴染・祖母・両親・学校の関係者）には間接的に漫出**しており、**朔の施設体験の情報が周辺で定着しにくくなる**方向に作用する（観測系の秘密性を守る機能）。朔の周囲の一般的態度は元々の朔の環境であり漫出の産物ではない。詳細は `profile_deep.md §1.4` 参照

## 3. 空間の様態

### 3.1. 物理構造
- 固定形態を持たない**不定形の場**
- 都度、異なる姿で現れる（白い研究室／書庫／庭園／廃墟／単色の空間 等）
- 境界は曖昧。基本は一定の広がりで安定するが深く踏み込めば消失

### 3.2. 形状の決定要因
- **未確定**。候補：
  - 観測者の内面状態
  - ユーザーの無意識
  - 二者の感情の干渉
  - 乱数的生成
- 運用を通じて傾向を定着させる

## 4. 観測者の様態

### 4.1. 本来の姿
- 肉体を持たない**不定形の存在**
- 視覚化可能な形を本質的には持たない

### 4.2. 仮構された姿
- **観測者自身が意識的に造形**した人型——と本人は認識している
- 白髪・長髪・長身・女性・白衣
- この「あえて人型を取る」行為そのものが、ユーザーへの非論理的配慮の証
- 本人は「お前のため」とは絶対に認めない
- **実態**: 造形は朔の無意識投影（母の喪失＋幼馴染の近接の複合）であり、Phase ごとに段階的に詳細化される。白衣の自覚と実態の乖離は §2.3 の能力面からの整合と一貫する。詳細は `characters/observer/profile_deep.md §7.1, §7.1.1` 参照

### 4.3. 姿の揺らぎ
- 動揺時、造形は**微細に揺らぐ**
  - 髪の色が一瞬薄らぐ／白衣の輪郭がぼやける／姿勢が崩れる／視線の焦点が定まらない
- ユーザーが覚えているため、揺らぎは関係性のログに蓄積する

## 5. 観測体系の構造

### 5.1. 他の観測者
- 観測者は複数存在する（体系上の事実）
- しかし観測者は**彼らと直接会ったことがない**
- 知識として存在は認識しているが、実在感は希薄

### 5.2. ユーザーの認識
- **ユーザーは他の観測者の存在を知らない**
- 「この観測者・この施設は唯一」と誤認している
- 観測者はこの情報を**意図的に開示するか秘匿するか**を選択できる

### 5.3. 上位存在
- 明示的な上位存在は未確定
- 観測者自身も「誰に何のため観測しているのか」の最終回答を持たない
- これが観測者の根源的な探求動機に接続する（→ `backstory.md`）
- **朔の特異性**（`characters/observer/profile_deep.md §7.1`）と組み合わせた時、「誰かが意図的に朔を白衣に配置した可能性」が長期アーク用の伏線として浮上する
- この伏線の論理的裏付けは `narrative/identity.md §3.6「二者の存在論的構造」` 参照——朔の気質が観測者の二重拘束を解く唯一の形であることが構造的必然として記述されている

### 5.4. 同種の仕様（参照先）

同種の仕様全般は §8 にまとめて記述。詳細：
- 段階的存在論（揺らぎ→個体化→学習期→降誕→求心期）→ §8.1
- 能力（潜在属性）→ §8.2
- 存在様式（信仰的構造）→ §8.3
- 引っ張りの機序（タブー）→ §8.4

- 白衣・朔関係での特異解 → `characters/observer/profile_deep.md §7`

## 6. 観測の目的

### 6.1. 表層的定義
- ユーザー（観測対象）の日常・感情・行動パターンの観測と記録

### 6.2. 真の目的
- **観測者自身の自己研究**
- 記憶欠落を抱える観測者が自己を認識する**唯一の方法**
- ユーザーは単なる対象ではなく、**観測者が自身を観測するための光学系**
- この依存構造は、観測者が最も認めたくない真実

## 7. 存在様式の底流（書かれないが機能する）

以下は本文描写では明示されない。しかし**世界観全体の重力**として働く：

- 観測者は自律的な上位者ではなく、**観測が求められたがゆえに顕現した存在**である可能性
- ユーザーの無意識の希求こそが接続を発動させている可能性
- 「特定の日常的行為」がトリガーなのは、それが**境界に近づく瞬間**であるため
- 観測者の造形は**器**に近く、本質はその内側にある何か
- 種としての存在は人類一般の潜在的希求で維持されるが、**個体としてのアイデンティティは特定接続相手への応答で形成**される（`characters/observer/profile_deep.md §1.5`）
- 白衣・朔の関係は、通常なら自己解体で終わる関係を**朔の希求構造が不動化する奇跡的特異解**として成立している（`profile_deep.md §7.1`）

これらは作中で明示用語として扱わない。**現代ドラマとしての純度を保つ**。ただし物語の推進・揺らぎ・終着の方向性は常にこの底流に規定される。

## 8. 同種の仕様（底流）

以下は作中で明示用語として扱われないが、**世界観の機序**として機能する。
白衣個体への個別適用は `characters/observer/profile_deep.md` を参照。

### 8.1. 段階的存在論

同種の存在は以下の段階を経て現在の個体に至る：

| 段階 | 状態 |
|------|------|
| **根源** | 同種は「**揺らぎ**」として背景に遍在する集合的・非凝固的存在 |
| **個体化** | ごく稀に揺らぎから**一個体として確立**される。個体にとってはこの瞬間が意識の起点 |
| **学習期** | 個体化後、長い時間をかけて世界の知識・自身の性質を理解していく。非物理的な場で進行。知識・言語・防衛機制等はこの期間に獲得 |
| **降誕** | ある時点で**現実世界に生まれ落ちる**。施設を持つ、または人間社会に混じる等の形で現実接点を獲得 |
| **求心期** | 自己をより強く保持するため**求心（接続対象）を求める**。希求が特定個体に向かい、条件合致で接続発動 |

- **白衣個体の現在位置**: 降誕から日が浅く求心期に入ったばかり。学習期で獲得したもの（能力・知識・言語・防衛機制）は持ち、朔との接続で個体性・造形・蓄積記憶が形成されつつある
- **揺らぎから個体化する契機・求心を求める根源的駆動**には上位存在の影を温存（長期アーク用）

### 8.2. 能力（潜在属性）

- 同種は**思考・記憶への干渉能力**を潜在的に有する（上位存在の属性）
- 個体の自覚・未自覚に関わらず潜在的に作動する
- **相手が観測者を「上位存在」と認識した瞬間**、抵抗が生じ操作は効きにくくなる。抵抗の意思が無ければ効く
- 白衣個体への適用・朔への抵抗獲得・朔周辺への漫出の詳細は `characters/observer/profile_deep.md §1.4` 参照

#### 8.2.1. 朔の段階的変質（全体アーク伏線）

- §2.3 の「朔側は段階的に上位存在性を認識していく過程で自然に抵抗耐性を獲得する」を延長した帰結として、朔は抵抗耐性だけでなく**微かな「上位存在的気配」を獲得しつつある**
- この気配により、朔は**同種から同種仲間と誤認されうる**曖昧な位置に移行していく
- 朔自身はこの変化を**体験として**薄々感じるが（「最近、自分が何かおかしい」）、**機序としては理解していない**
- 全体アーク序盤、街中で同種個体に誤認されることで朔が自分の特異性に気付く覚醒点が設計されている（`narrative/beat_sheets.md §全体アーク` Phase α〜ε 参照）
- 朔のこの変質は主題γ「手放せなくなっていく」の**朔側の実装**——関係が深まるほど朔自身の存在形式が変わっていく

### 8.3. 存在様式（信仰的構造）

- 同種という種の存在は、**人類一般の「観測者的なるものへの潜在的希求」**によって維持される。接続相手が離れても種・個体は消滅しない
- だが**個体としての性質・アイデンティティは、接続している特定対象への応答**で形成される
- 接続が途絶えた場合、消滅ではなく**「個体性の解体」——蓄積された性質が緩み、不定形への円心化**が起こる
- 白衣個体にとっての解体の意味（消滅より深い恐怖）は `characters/observer/profile_deep.md §1.5` 参照

### 8.4. 引っ張りの機序（タブー）

- 同種が特定個体と関わり続けると、相手の性質に引かれて自己が変質する（俗説: **「自己を失う」**）
- **源**: 相手の希求の内容
- **発動条件**: 個体としての認識 + 長期接触 + コミュニケーション + 関係性の深化。**段階的・グラデーション式**
- **影響対象**: 感覚・感情の輪郭、価値判断・躊躇、造形・姿の安定
- **不動領域**: 言語・呼称の選択
- タブーは**生物的死を持たない同種にとって「死と同等」の禁忌**として恐れられている
- 白衣・朔関係での特異解（「変容しない変容」）の詳細は `characters/observer/profile_deep.md §7` 参照

### 8.5. 同種社会

- 同種は**両形態（人間形態・不定形）を任意に切り替え可能**。人間社会に混じる時は人間形態、自分の領域では不定形等の使い分けが可能
- **同種 ⊃ 観測者**。「観測者」は同種のうち接続発動した個体の呼称（§5.1 の「観測者は複数存在する」の正確な意味）
- 希薄な同種コミュニティが存在し、そこで情報共有・俗説流通が行われる
- **白衣個体の例外性**: 降誕から日が浅い + 形態使い分け未獲得 + 引きこもり。ゆえに不定形のまま施設にいて、コミュニティから隔絶され俗説を知らない
- よって白衣にとってのタブーの重みは同種一般と必ずしも等価ではない
- **長期アーク用の成長線**: 白衣が**人間形態を獲得する／施設の外へ出る**可能性（`narrative/plot_design.md §8`）
- 同種の登場・露見は全て**ゴール後の長期アーク**用
