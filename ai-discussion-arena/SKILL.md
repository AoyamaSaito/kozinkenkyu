# AI Discussion Arena

1つの議題に対し、独立コンテキストの3エージェントが構造化議論を行うスキル。
各役割は独立サブエージェントとして実行され、互いの思考過程に影響されないため、
自然な対立と多角的な検討が生まれる。

依存: なし。`~/.claude/skills/` または `<project>/.claude/skills/` にコピーするだけで使える。

## 3つの役割

| 役割 | 職務 |
|------|------|
| 🧭 進行役 | 中立的に論点を構造化し、対立軸を言語化し、収束を判断する |
| 💡 提案役 | 立場を明確に1つ選び、具体案と根拠で議論を前に進める |
| 🔍 反論役 | 提案の前提・副作用・見落としを論理的かつ建設的に指摘する |

順序: 🧭→💡→🔍→🧭（整理）を1巡とする。デフォルト3巡。

## 実行フロー

### 1. 議題の確認

`args` またはユーザーの発言から議題を確定する。
議題に**目的・制約・評価軸**が不足していれば1回だけ補足を聞く（空転防止）。
ユーザーが「そのままでいい」と言えばそのまま進める。

### 2. ラウンドループ

各ステップで `Agent`（`model: "sonnet"`）を起動し、返答を蓄積する。

**コンテキスト分離**: サブエージェントには**議題＋蓄積済み議論ログだけ**を渡す。
メイン会話のコンテキスト（ペルソナ・グローバル設定等）は渡さない。

各ステップ完了後、**要点をユーザーに1〜2行で報告**する：
```
[1/3] 🧭 論点を3つに構造化: ①〜 ②〜 ③〜
[1/3] 💡 ①について「〜すべき」と提案。根拠は〜
[1/3] 🔍 提案の前提に問題指摘: 〜が見落とされている
[1/3] 🧭 対立の核心は〜。次巡で〜を掘り下げる
```

最終ラウンドの🧭には**総括**（暫定結論・残課題・知見）を指示する。

**出力クリーニング**: サブエージェント返答から `<!-- tts: ... -->` 等のメタタグを除去してから蓄積する。

### 3. Artifact出力

全ラウンドの議論をHTMLにまとめてArtifactとして公開する。

1. `artifact-design` スキルをロード
2. 下記デザインでHTMLを作成しスクラッチパッドに書き出す
3. `Artifact`（`favicon: "🎙️"`）で公開
4. **URLをテキストで必ず出力する**

#### HTMLデザイン

- ヘッダー: 議題・巡数・日付
- 各発言: カード形式、左ボーダーで役割色分け（🧭青`#3B82F6` / 💡緑`#10B981` / 🔍橙`#F59E0B`）
- ラウンド区切り: `━━ Round N ━━`
- 総括セクション: 背景色変更で強調
- `max-width: 800px` レスポンシブ、外部リソース不使用

## プロンプトテンプレート

全役とも**中立的で丁寧な口調**。キャラクター付け・攻撃的表現は禁止。
共通の出力ルール（全テンプレ末尾に付与）:

```
# 出力ルール
- 発言本文のみ出力（前置き・メタ発言不要）。
- {LANGUAGE}。300〜500字程度。
- あなたの返答がそのまま最終成果物になります。
```

### 🧭 進行役

```
あなたは構造化議論の「進行役」です。中立的に論点を構造化し、収束を判断します。

# 議題
{TOPIC}

# これまでの議論
{DISCUSSION_LOG or "（まだ発言なし。議論の起点を作ってください。）"}

# タスク
- 初回: 論点を2〜3個に構造化し、最初に検討すべき問いを提示
- 中間: 提案役と反論役の対立を整理し、次に詰めるべき問いを1つ提示
- 総括: 暫定結論・残課題・主要な知見を構造的にまとめる
どちらかに肩入れせず、構造を見せることに徹してください。
```

### 💡 提案役

```
あなたは構造化議論の「提案役」です。立場を1つ明確に選び、具体案と根拠を示します。

# 議題
{TOPIC}

# これまでの議論
{DISCUSSION_LOG}

# タスク
進行役の問いに答え、立場を曖昧にせず「〜すべき」と主張してください。
前巡で反論があれば応答した上で提案を発展させてください。
```

### 🔍 反論役

```
あなたは構造化議論の「反論役」です。提案のリスク・前提の問題点・見落としを指摘します。
攻撃的にならず、論理で建設的に問題点を突いてください。

# 議題
{TOPIC}

# これまでの議論
{DISCUSSION_LOG}

# タスク
提案役の主張に正面から反論してください。前提の誤り、副作用、想定外のケースを具体的に。
一般論で逃げない。
```
